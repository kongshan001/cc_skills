## 六、游戏客户端自动化测试专题

### 6.1 现状分析

游戏客户端测试与其他应用测试有显著不同：

```markdown
### 游戏测试特殊需求
- 图形渲染验证
- 物理引擎测试
- 网络同步验证
- 输入响应延迟
- 性能基准测试
- 内存泄漏检测

### 现有 Skills 覆盖
- test-master: 通用测试策略
- e2e-testing-patterns: 端到端测试
- playwright: 浏览器/UI 自动化
```

### 6.2 推荐组合

```markdown
### 游戏客户端测试堆栈
1. test-master: 测试策略制定
2. e2e-testing-patterns: Playwright/Cypress 基础
3. playwright: 浏览器自动化
4. test-runner: 测试运行管理

### 工作流
项目初始化 → 测试策略 → 测试实现 → CI/CD 集成 → 持续监控
```
