## 📝 报告规范

每个技能的调研报告包含以下部分：

1. **基本信息**: 名称、版本、作者、创建时间、更新时间
2. **技能描述**: 技能的核心功能和用途
3. **功能列表**: 详细的功能列表
4. **安装方式**: 安装命令和步骤
5. **推荐安装评估**: 本地环境 vs ECS 服务器环境的评估
6. **依赖要求**: 需要的依赖和配置
7. **使用场景**: 适合的使用场景
8. **标签**: 相关标签信息
