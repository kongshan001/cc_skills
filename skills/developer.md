# Developer 通用开发者技能调研报告

## 1. 背景需求

通用开发技能是最基础的编程指导能力，适用于各种编程语言和场景。作为入门级技能，帮助开发者解决日常编程问题。

## 2. 目标

提供通用开发能力：
- 代码编写指导
- 问题诊断
- 最佳实践
- 代码审查

## 3. 设计方案

### 核心能力
- **代码编写**: 多语言编程指导
- **问题诊断**: Bug 排查和修复
- **最佳实践**: 编码规范和模式
- **代码审查**: 代码质量检查

### 技术特点
- 语言无关
- 场景广泛
- 基础能力

## 5. 效果展示

典型应用场景：
