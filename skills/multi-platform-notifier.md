# multi-platform-notifier (多平台通知)

## 技能描述

统一接口发送企业微信、钉钉、飞书通知，支持配置管理、消息模板和发送历史查询。

## 功能列表

- 企业微信通知
- 钉钉通知
- 飞书通知
- 统一发送接口
- 配置管理
- 消息模板
- 发送历史查询

## 安装方式

```bash
clawhub install multi-platform-notifier
```

## 推荐安装评估

- **本地部署**: ⭐⭐⭐⭐⭐ (通知聚合必备)
- **ECS 部署**: ⭐⭐⭐⭐⭐ (适合监控告警)
