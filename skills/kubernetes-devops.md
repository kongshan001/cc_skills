# Kubernetes DevOps

> 技能Slug: kubernetes-devops  
> 版本: 1.0.0  
> 作者: wpank  
> 更新时间: 2026-02-26

## 技能描述

WHAT: Kubernetes manifest generation - Deployments, StatefulSets, CronJobs, Services, Ingresses, ConfigMaps, Secrets, and PVCs with production-grade security and health checks.

生产级 K8s 清单生成技能，支持多种资源类型和安全配置。

## 功能列表

- Deployment 生成
- StatefulSet 生成
- CronJob 生成
- Service 配置
- Ingress 配置
- ConfigMap & Secret
- PVC 持久化存储
- 安全上下文配置
- 健康检查配置

## 安装方式

```bash
clawhub install kubernetes-devops
```

## 推荐安装评估

### 本地开发环境 ⚠️ 需谨慎

- 需要 minikube/kind 或远程集群
- 适合学习 K8s

### ECS/服务器 ✅ 推荐

- 生产环境部署
- 多环境配置管理

## 适用场景

1. K8s 清单生成
2. 生产环境部署
3. 多环境配置
4. 服务暴露配置
