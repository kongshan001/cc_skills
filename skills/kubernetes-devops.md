# Kubernetes DevOps

## 技能描述

Kubernetes 资源清单生成工具，支持 Deployment、StatefulSet、CronJob、Service、Ingress、ConfigMap、Secret 和 PVC，配置生产级安全性和健康检查。

## 功能列表

- Deployment 资源生成
- StatefulSet 资源生成
- CronJob 资源生成
- Service 和 Ingress 配置
- ConfigMap 和 Secret 管理
- PVC 持久化存储配置
- 生产级安全上下文配置
- 健康检查探针配置
- 多环境配置组织

## 安装方式

```bash
clawhub install kubernetes-devops
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐ | K8s 学习需要 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | K8s 部署必备技能 |

## 使用场景

- K8s 资源清单编写
- 微服务部署配置
- 多环境配置管理
- 生产环境安全加固
