## 2. 功能描述

Read and send email via IMAP/SMTP. Check for new/unread messages, fetch content, search mailboxes, mark as read/unread, and send emails with attachments.

## 3. 核心功能

- 📬 读取邮件 (IMAP)
- 📤 发送邮件 (SMTP)
- 🔔 新邮件检查
- 🔍 邮箱搜索
- ✅ 标记已读/未读
- 📎 附件支持

## 4. 安装方式

```bash
clawhub install imap-smtp-email
```

## 5. 推荐安装评估

### 本地安装 ⭐⭐⭐⭐ (推荐)

- **适用场景**: 邮件自动化
- **优势**: 
  - 邮件收发自动化
  - 附件处理
  - 搜索功能
- **要求**: 需要邮箱账号配置

### 阿里云 ECS ⭐⭐⭐ (一般推荐)

- **适用场景**: 定时邮件任务
- **优势**: 
  - 自动化报告发送
  - 定时任务集成
- **注意**: 
  - 需要 SMTP/IMAP 支持
  - 注意邮件发送限制

## 6. 适用场景

- 自动化报告发送
- 邮件通知系统
- 邮件聚合阅读
