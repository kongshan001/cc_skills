## 技能描述

Database design skills for creating efficient, normalized, and scalable database schemas. Use when designing new databases, optimizing existing schemas, or creating ER diagrams.

数据库设计技能，用于创建高效、规范、可扩展的数据库模式。适用于设计新数据库、优化现有模式、创建 ER 图。

## 功能列表

- 需求分析
- 概念模型设计 (ER 图)
- 逻辑模型设计
- 物理模型设计
- 范式规范化
- 索引优化
- 表关系设计
