# Custom Tools

## 技能描述

提供 18 个 OpenClaw 工具，用于任务管理、技能加载、代码导航、配置管理和 MCP 客户端交互。

## 功能列表

- 任务管理
- 技能加载
- 代码导航
- 配置管理
- MCP 客户端交互

## 安装方式

```bash
clawhub install customtools
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐ | OpenClaw 扩展 |
| ECS | ⭐⭐⭐ | 定制工具 |

## 适用场景

- OpenClaw 扩展
- 任务流程定制
- 工具链构建
