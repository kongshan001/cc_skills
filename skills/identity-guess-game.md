# Identity Guess Game

## 基本信息

- **Slug**: identity-guess-game

- **版本**: 1.0.0
- **更新时间**: 2026-03-31

## 技能描述

身份猜猜猜 - 多人互动推理游戏。每位玩家被分配一个秘密身份，通过3轮线索描述暗示自己的身份，最终猜测其他玩家的身份。

## 功能列表

- 多人身份分配
- 3轮线索描述环节
- 身份猜测机制
- 互动推理流程
- 游戏裁判模式
