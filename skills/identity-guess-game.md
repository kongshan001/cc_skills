# Identity Guess Game

## 技能概述

- **Slug**: identity-guess-game
- **名称**: Identity Guess Game (身份猜猜猜)
- **作者**: loongfay
- **版本**: 1.0.0
- **更新时间**: 2026-03-31

## 背景需求

多人互动推理游戏是一种有趣的社交游戏形式，需要一个可以在 AI 辅助下进行的游戏系统。

## 目标

身份猜猜猜 - 多人互动推理游戏。每位玩家被分配一个秘密身份，通过3轮线索描述暗示自己的身份，最终猜测其他玩家的身份。

## 功能列表

- 多人身份分配
- 3轮线索描述
- 身份猜测
- 互动推理
- 游戏裁判
