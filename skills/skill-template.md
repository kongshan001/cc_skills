# Skill Template - 技能模板生成器

## 技能描述

**Skill Template** 是 OpenClaw 技能模板生成器。创建技能脚手架、验证结构、增强 SKILL.md、生成命令框架、提示词、发布检查清单。

- **最新版本**: 2.0.1
- **作者**: ckchzh
- **更新时间**: 2026-04-21
- **标签**: chinese, productivity, template, generator

## 功能列表

- 技能脚手架创建
- SKILL.md 结构验证
- 增强现有技能文档
- 命令框架生成
- 发布检查清单

## 安装方式

```bash
clawhub install skill-template
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 强烈推荐 | 创建技能必备 |
| ECS 服务器 | ✅ 适合 | 适合自动化流程 |

## 适用场景

- 新手创建第一个技能
- 标准化技能结构
- 技能发布前检查
