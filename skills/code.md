## 技能描述

Coding workflow with planning, implementation, verification, and testing for clean software development.

编码工作流技能，提供规划、实施、验证和测试功能，支持干净的软件开发流程。

## 功能列表

- 规划 (Planning)
- 实施 (Implementation)
- 验证 (Verification)
- 测试 (Testing)
- 代码审查
- 开发流程优化

## 安装方式

```bash
# 使用 ClawHub 安装
npx clawhub@latest install code

# 或全局安装 ClawHub 后安装
npm i -g clawhub
clawhub install code
```

## 推荐安装评估

### 本地安装 ⭐⭐⭐⭐⭐
- 适合场景：本地开发环境、日常编码任务
- 优势：低延迟、完全离线可用、隐私保护
- 要求：Node.js 环境

### ECS 安装 ⭐⭐⭐⭐
- 适合场景：远程开发服务器、团队协作环境
- 优势：可集中管理、团队共享
- 要求：ECS 实例需安装 Node.js

## 优缺点分析

### 优点
- 完整的工作流覆盖
- 适合多种编程语言
- 强调代码质量和测试

### 缺点
- 依赖于具体开发环境配置
- 需要一定的学习成本
