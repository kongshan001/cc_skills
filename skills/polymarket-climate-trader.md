# Polymarket Climate Trader — 气候预测市场交易

## 背景需求

交易气候相关预测市场（天气极端事件、气候里程碑、自然灾害、农业产出）需要专业工具。

## 目标

交易 Polymarket 上关于天气极端事件、气候里程碑、自然灾害和农业结果的预测市场。

## 设计方案

- 气候事件市场监控
- 自动化交易执行
- 风险控制

## 本地部署

```bash
npm i -g clawhub
clawhub install polymarket-climate-trader
```

## 推荐安装

**ECS**：✅ 推荐 - 需7x24运行

## 效果展示

- 市场数据看板
- 自动交易

## 优缺点分析

**优点**：
- 专注气候赛道

**缺点**：
- 预测市场风险
