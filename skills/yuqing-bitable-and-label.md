# yuqing-bitable-and-label (舆情 Bitable 与标签)

## 技能描述

Incrementally sync data from XiaoAi API to Feishu Bitable and optionally auto-label records with machine-based type and sentiment annotations.

增量同步小爱 API 数据到飞书多维表格，并可选地使用机器学习进行类型和情感自动标注。

## 功能列表

- 小爱 API 数据增量同步到飞书 Bitable
- 机器学习类型标注 (type labeling)
- 情感分析标注 (sentiment annotation)
- 品牌安全检测
- 竞品提及检测
- 设备类型识别
- 内容安全审核

## 安装方式

```bash
clawhub install yuqing-bitable-and-label
```
