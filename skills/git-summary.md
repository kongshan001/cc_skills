## 技能描述

Get a quick summary of the current Git repository including status, recent commits, branches, and contributors.

快速获取当前 Git 仓库摘要，包括状态、最近提交、分支和贡献者。

## 功能列表

- 仓库状态摘要
- 最近提交查看
- 分支信息
- 贡献者列表
- 快速概览
