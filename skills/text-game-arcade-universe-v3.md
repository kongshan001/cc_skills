# text-game-arcade-universe-v3 - 综合性 ASCII 文字游戏大厅

## 技能描述

综合性 ASCII 文字游戏大厅，支持严格规则渲染和裁判多种棋类游戏：五子棋、围棋、井字棋、黑白棋、四子棋、海战棋、中国象棋、国际象棋、扫雷、数独、2048、推箱子、文字迷宫，以及接龙、猜词、猜数、Lights Out、Nim、汉诺塔等小游戏。支持强触发词、统一大厅、标准棋盘模板、合法性校验、难度、悔棋与裁判模式。

## 功能列表

- 五子棋 / 围棋 / 井字棋 / 黑白棋 / 四子棋
- 海战棋 / 中国象棋 / 国际象棋
- 扫雷 / 数独 / 2048 / 推箱子 / 文字迷宫
- 接龙 / 猜词 / 猜数 / Lights Out / Nim / 汉诺塔
- 合法性校验与裁判模式
- 难度分级与悔棋功能

## 安装方式

```bash
clawhub install text-game-arcade-universe-v3
```
