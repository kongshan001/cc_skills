# 文字游戏大观园 (text-game-arcade-universe-v3)

## 背景需求

玩家希望在聊天场景中快速启动各类经典文字/棋盘游戏，但每款游戏单独开发成本高。需要一个统一的游戏大厅，同时支持多种游戏。

## 目标

构建综合性 ASCII 文字游戏大厅，统一触发词入口，提供五子棋、围棋、井字棋等棋类及多种休闲小游戏。

## 设计方案

- **支持游戏**: 五子棋、围棋、井字棋、黑白棋、四子棋、海战棋、中国象棋、国际象棋、扫雷、数独、2048、推箱子、文字迷宫、接龙、猜词、猜数、Lights Out、Nim、汉诺塔等
- **核心特性**:
  - 强触发词统一入口
  - 标准棋盘 ASCII 模板
  - 规则合法性校验
  - 难度分级
  - 悔棋与裁判模式

## 本地部署

```bash
clawhub install text-game-arcade-universe-v3
```

## 效果展示

- 统一大厅入口，一个技能玩转20+游戏
- 严格规则校验，AI 充当公正裁判
- ASCII 渲染，终端友好

## 优缺点分析

✅ 优点:
- 游戏种类丰富，一站式体验
- 规则严格，裁判公正
- 纯文字界面，兼容性好

❌ 缺点:
- ASCII 图形对视觉不够友好
- 复杂游戏（如围棋）体验有限

## 平替对比

| 特性 | text-game-arcade-universe-v3 | 单游戏技能 |
|------|------------------------------|------------|
