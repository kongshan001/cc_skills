# SQL Toolkit

## 技能描述

Query, design, migrate, and optimize SQL databases. Use when working with SQLite, PostgreSQL, or MySQL — schema design, writing queries, creating migrations, indexing, backup/restore, and debugging slow queries. No ORMs required.

**所有者**: gitgoodordietrying  
**创建时间**: 2026-02-03  
**最新版本**: 1.0.0  
**ClawHub Slug**: `sql-toolkit`

## 功能列表

- SQL 查询编写和优化
- 数据库 Schema 设计
- 数据库迁移（Migrations）
- 索引优化
- 备份和恢复
- 慢查询调试
- 支持 SQLite、PostgreSQL、MySQL
- 无需 ORM

## 安装方式

```bash
clawhub install sql-toolkit
```

## 推荐安装评估

### 本地环境
- ✅ **强烈推荐** - 数据库开发必备
- 适合后端开发者和 DBA
- 支持多种数据库
- 资源占用低

### ECS/云服务器
- ✅ **强烈推荐** - 生产数据库管理必备
- 适合数据库优化和维护
- 需要数据库访问权限
- 注意生产环境安全操作

## 使用场景

1. 数据库 Schema 设计
2. 复杂 SQL 查询编写
3. 性能优化和索引设计
4. 数据库迁移脚本生成
5. 慢查询分析
6. 数据库备份恢复
7. 多数据库统一管理
