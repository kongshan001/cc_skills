# Senior Dev

## 技能描述

生产级开发工作流，包含TODO追踪、Graphite PRs、GitHub issues、Vercel部署检查和短信通知。适用于开始新任务、修复bug、实现功能或任何需要进度追踪和代码审查的开发工作。

## 核心功能

- TODO任务追踪
- GitHub Issues集成
- Graphite PRs
- Vercel部署检查
- 短信通知

## 安装方式

```bash
clawhub install senior-dev
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐⭐ 适用

**理由**: 专业级开发工作流，适合有规范流程的开发团队。
