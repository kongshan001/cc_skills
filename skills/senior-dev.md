# Senior Dev

> 生产级开发工作流

## 技能描述

生产开发工作流，包含 TODO 追踪、Graphite PRs、GitHub issues、Vercel 部署检查和短信通知。适用于新任务开发、Bug 修复、功能实现等场景。

## 核心功能

1. **TODO 追踪** - 任务进度跟踪
2. **Graphite PRs** - PR 管理
3. **GitHub Issues** - Issue 追踪
4. **Vercel 部署检查** - 部署验证
5. **短信通知** - 重要事项通知

## 安装方式

```bash
clawhub install senior-dev
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 资深开发者必备 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 团队协作流程 |
| 个人项目 | ⭐⭐⭐⭐ | 提升工程质量 |

### 适用人群

- 资深开发者
- 追求工程质量的团队
- 需要完整开发流程的项目
