# Web Screenshot (网页截图工具)

## 技能概述

**Slug**: `web-screenshot`  
**作者**: jiaonvmo-maker  
**版本**: 1.1.3  
**更新日期**: 2026-03-21

## 背景需求

任意 URL 全页面截图 + PDF 导出工具。当用户要求截取网页、保存网页快照、截图存档、做 QA 对比、导出 PDF 时使用。

## 功能列表

1. 全页面截图（PNG/JPG）
2. PDF 导出
3. 支持中文网站（百度/知乎/微信公众号/小红书等）
4. 自动等待 JS 渲染

## 安装方式

```bash
clawhub install web-screenshot
```

## 推荐安装评估

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 本地开发 | ⭐⭐⭐⭐ | 需要无头浏览器 |
| ECS 服务器 | ⭐⭐⭐⭐ | 支持 Docker 部署 |

## 优缺点分析

**优点**:
- 支持中文网站
- 自动 JS 渲染等待
- PDF 导出功能

**缺点**:
- 需要浏览器环境
- 占用资源较多
