# YDC AI SDK Integration

## 技能描述

将 Vercel AI SDK 应用与 You.com 工具集成（web search、AI agent、content extraction）。用于开发者提及 AI SDK、Vercel AI SDK、generateText、streamText 或 You.com 与 AI SDK 集成时。

## 功能列表

- Web Search 工具集成
- AI Agent 工具集成
- 内容提取工具
- Vercel AI SDK 集成模板

## 安装方式

```bash
clawhub install ydc-ai-sdk-integration
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | Vercel 项目开发 |
| ECS | ⭐⭐⭐⭐ | 生产环境部署 |

## 适用场景

- 构建 AI 驱动的 Web 应用
- 集成 You.com 搜索能力
- 流式 AI 响应处理
