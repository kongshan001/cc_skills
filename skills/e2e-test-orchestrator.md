# E2E Test Orchestrator

> 端到端（E2E）测试编排与执行。基于 Playwright/Cypress 实现自动化脚本，支持源码定位元素和图像识别兜底，自动修复脚本问题。

## 技能概述

E2E Test Orchestrator 提供完整的端到端测试解决方案，基于 Playwright/Cypress 实现自动化。

## 功能列表

- 测试用例设计
- Playwright 自动化脚本
- Cypress 自动化脚本
- 源码优先元素定位
- 图像识别兜底
- 自动修复定位器
- 智能等待策略
- 结构化测试报告

## 安装方式

```bash
clawhub install e2e-test-orchestrator
```
