## 技能描述

Use when designing database schemas, writing migrations, optimizing SQL queries, fixing N+1 problems, creating indexes, setting up PostgreSQL, configuring EF Core, implementing caching, partitioning tables, or any database performance question.

数据库操作技能，用于数据库模式设计、迁移编写、SQL 查询优化、修复 N+1 问题、创建索引、PostgreSQL 设置、EF Core 配置、实现缓存、表分区等。

## 功能列表

- 数据库模式设计
- 迁移编写
- SQL 查询优化
- N+1 问题修复
- 索引创建
- PostgreSQL 设置
- EF Core 配置
- 缓存实现
- 表分区

## 安装方式

```bash
# 使用 ClawHub 安装
npx clawhub@latest install database-operations

# 或全局安装 ClawHub 后安装
npm i -g clawhub
clawhub install database-operations
```

## 推荐安装评估

### 本地安装 ⭐⭐⭐⭐
- 适合场景：本地开发、数据库学习
- 优势：本地实验、快速反馈
- 要求：Node.js 环境 + 数据库客户端

### ECS 安装 ⭐⭐⭐⭐⭐
- 适合场景：生产数据库管理、数据库性能优化
- 优势：可连接生产数据库、批量操作
- 要求：ECS 实例 + 数据库连接

## 优缺点分析

### 优点
- 多种数据库支持
- 覆盖开发和运维场景
- 性能优化能力强

### 缺点
- 需要数据库环境
- 某些操作需要 DBA 权限
