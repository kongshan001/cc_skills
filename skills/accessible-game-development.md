# Accessible Game Development

| 项目 | 内容 |
|------|------|
| **技能名称** | accessible-game-development |
| **所有者** | wenwei020 |
| **版本** | 1.0.0 |
| **更新时间** | 2026-04-06 |

## 技能描述

开发支持屏幕阅读器（如 ZHENGDU、NVDA、JAWS）的无障碍 HTML 游戏，支持盲人和低视力用户。

## 功能列表

- 无障碍 HTML 游戏开发
- 屏幕阅读器适配（NVDA、JAWS、ZHENGDU）
- 无障碍标准遵循
- 盲人/低视力用户支持

## 安装方式

```bash
clawhub install accessible-game-development
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| **本地开发** | ⭐⭐⭐⭐⭐ | 适合无障碍游戏开发 |
| **ECS服务器** | ⭐⭐⭐ | 需要无障碍测试环境 |

## 使用场景

- 无障碍游戏开发
- 教育类游戏适配
- 特殊人群游戏
