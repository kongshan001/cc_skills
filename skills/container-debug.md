# Container Debug

| 项目 | 内容 |
|------|------|
| **技能名称** | container-debug |
| **所有者** | gitgoodordietrying |
| **版本** | 1.0.0 |
| **更新时间** | 2026-02-26 |

## 技能描述

调试运行中的 Docker 容器和 Compose 服务。用于检查容器日志、exec 进入容器、诊断网络问题、检查资源使用、调试多阶段构建、排查健康检查、修复 Compose 依赖。

## 功能列表

- 容器日志检查
- 容器 exec
- 网络诊断
- 资源使用检查
- 多阶段构建调试
- 健康检查排查
- Compose 依赖修复

## 安装方式

```bash
clawhub install container-debug
```
