# Cognitive Tools

## 技能描述

整合李继刚原创的 11 个认知工具，实现逻辑分析、问题挖掘、抽象表达、多角度思考和公文写作等深度认知任务。

## 功能列表

- 逻辑分析
- 问题挖掘
- 抽象表达
- 多角度思考
- 公文写作
- 深度认知任务

## 安装方式

```bash
clawhub install cognitive-tools
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
