# Web-MCP

## 技能描述

WebMCP - 使 AI agents 能通过结构化工具与 Web 应用交互。为 Next.js/React 应用实现 WebMCP 标准，支持工具注册和调用。

## 功能列表

- WebMCP 标准实现
- Next.js/React 集成
- 工具注册系统
- 结构化工具调用

## 安装方式

```bash
clawhub install web-mcp
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | Next.js 项目 |
| ECS | ⭐⭐⭐⭐ | 生产部署 |

## 适用场景

- 构建 AI 可交互 Web 应用
- 暴露工具给 AI agents
- Next.js 工具集成
