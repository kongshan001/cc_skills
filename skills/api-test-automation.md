# API Test Automation

## 技能描述

API接口测试自动化工具，支持REST/GraphQL，包含接口测试、性能测试、契约测试、Mock服务等功能 | API Test Automation for REST/GraphQL with performance, contract testing and Mock services.

**所有者**: kaiyuelv  
**创建时间**: 2026-03-20  
**最新版本**: 1.0.0  
**ClawHub Slug**: `api-test-automation`

## 功能列表

- REST API 测试
- GraphQL API 测试
- 性能测试（负载测试、压力测试）
- 契约测试（Contract Testing）
- Mock 服务生成
- 测试报告生成
- 自动化测试脚本
- CI/CD 集成

## 安装方式

```bash
clawhub install api-test-automation
```

或指定版本：

```bash
clawhub install api-test-automation --version 1.0.0
```

## 推荐安装评估

### 本地环境
- ✅ **强烈推荐** - API 开发和测试必备
- 适合前端和后端开发者
- 需要 Python 或 Node.js 环境
- 资源占用低

### ECS/云服务器
- ✅ **强烈推荐** - 适合 CI/CD 流水线集成
- 可集成到自动化测试平台
- 适合性能测试场景
- 需要配置测试环境隔离

## 使用场景

1. API 接口功能测试
