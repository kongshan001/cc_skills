# Devloop Agent Pack

> 多 Agent 协作开发系统

## 技能描述

完整的多 Agent 协作系统，用于产品驱动的开发循环。涵盖从产品发现到架构设计、开发的完整生命周期。

## 核心功能

1. **多 Agent 协作** - 多代理协作系统
2. **产品驱动开发** - Product-driven 开发循环
3. **产品发现** - Product discovery
4. **架构设计** - Architecture design
5. **完整开发周期** - 从需求到部署

## 安装方式

```bash
clawhub install devloop-agent-pack
```
