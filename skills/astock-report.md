# Astock Report — A股报告生成与推送

## 背景需求

A股投资者需要晨/晚报、盘中实时监控和财经周末要闻等报告，以辅助投资决策。

## 目标

自动生成并推送 A股报告，包含投资者情绪打分和 AI 后市展望。

## 设计方案

- 晨报 (cron 08:00)
- 晚报 (cron 20:00)
- 收盘小结 (cron 15:00)
- 盘中预警 (cron 每5分钟)
- 财经周末要闻 (cron 周五20:00)
- IPO周报 (cron 周六09:00)
- 投资者情绪打分（6维度，满分100）
- AI 后市展望

## 本地部署

```bash
npm i -g clawhub
clawhub install astock-report
```

## 推荐安装

**本地 / ECS**：✅ 推荐 - 定时任务需持续运行

## 效果展示

- 每日晨/晚报
- 实时预警推送
- 情绪分数仪表板

## 优缺点分析

**优点**：
- 全天候覆盖
- 情绪量化模型
- 多数据源整合

**缺点**：
- 仅支持 A股
