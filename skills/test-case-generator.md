## 技能描述

Automatically generate complete unit test code including imports, test cases, mocks, and assertions for Jest, Mocha, or Pytest from given code.

根据给定代码自动生成完整的单元测试代码，包括导入、测试用例、mock 和断言，支持 Jest、Mocha 或 Pytest。

## 功能列表

- 自动生成单元测试
- 测试导入生成
- 测试用例生成
- Mock 对象生成
- 断言生成
- Jest 支持
- Mocha 支持
- Pytest 支持
