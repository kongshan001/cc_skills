# The Flip Publish - 区块链翻转游戏

## 技能描述

$1 USDC entry. 14 coin flips. Get all 14 right, take the entire jackpot. Live on Solana devnet — continuous game, enter anytime.

## 功能列表

- USDC 入金 (1 USDC)
- 14次翻转竞猜
- Solana devnet 部署
- 持续游戏机制
- 奖金池奖励

## 安装方式

```bash
clawhub install the-flip-publish
```
