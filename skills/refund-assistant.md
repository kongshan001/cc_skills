# Refund Assistant - 退款助手

## 基本信息

- **技能名称**: refund-assistant
- **版本**: 1.0.0
- **作者**: harrylabsj
- **创建时间**: 2026-03-15
- **更新时间**: 2026-03-15

## 技能描述

Guide users through refund and return decisions for ecommerce orders, including reasons, evidence preparation, and platform-specific strategy.

指导用户完成电商订单的退款和退货决策，包括退款理由、证据准备和平台特定策略。

## 功能列表

- 退款和退货决策指导
- 退款理由分析
- 证据准备建议
- 平台特定策略（淘宝、京东、拼多多等）
- 退款流程指导
