# Local Skill Manager - 本地技能管理器

## 技能描述

**Local Skill Manager** 是本地 Skill 目录的集中管理工具（"Skill 管家"）。可以列出/查询已安装的 Skill、检查版本信息、使用标准模板创建新 Skill、删除不需要的 Skill。

- **最新版本**: 1.0.1
- **作者**: chenchen913
- **更新时间**: 2026-03-20
- **标签**: ai-development, cli-tools, developer-tools, environment-management, local-tools, scaffolding, skill-management, skill-manager, toolchain, version-check

## 功能列表

- 列出/查询已安装技能
- 版本信息检查
- 标准模板创建新技能
- 技能删除/移除
- 系统级/用户级技能管理

## 安装方式

```bash
clawhub install local-skill-manager
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 强烈推荐 | 本地技能管理必备 |
| ECS 服务器 | ✅ 适合 | 服务器技能管理 |

## 适用场景

- 技能清单管理
- 版本追踪
- 技能创建删除
- 环境清理
