## 技能描述

使用 pylsp 进行 Python 代码质量检查和 LSP 集成，提供代码诊断、补全、悬停提示和样式分析。

## 功能列表

- 代码诊断
- 智能补全
- 悬停提示
- 样式分析
- Python 代码质量检查

## 安装方式

```bash
clawhub install lsp-python
```
