## 背景需求

Python 代码质量检查和 LSP（语言服务器协议）集成是提升开发效率的关键。开发者需要代码诊断、补全、悬停提示和样式分析等功能。

## 目标

提供 Python 代码质量检查和 LSP 集成功能，使用 pylsp 实现代码诊断、补全、悬停提示和样式分析。

## 功能列表

1. **代码诊断** - 实时检测代码问题
2. **代码补全** - 智能代码补全
3. **悬停提示** - 悬停显示信息
4. **样式分析** - 代码风格检查
5. **代码质量** - 提升代码质量
