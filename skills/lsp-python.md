## 背景需求

Python 代码质量检查和语言服务集成需要专业工具。

## 目标

提供 Python 代码诊断、补全、悬停提示和风格分析。

## 功能列表

- 代码诊断 (Diagnostics)
- 代码补全 (Completion)
- 悬停提示 (Hover)
- 样式分析 (Style Analysis)
- 使用 pylsp
