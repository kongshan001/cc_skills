## 技能描述

Python code quality checking and LSP integration using pylsp. Provides code diagnostics, completion, hover tips, and style analysis.

使用 pylsp 进行 Python 代码质量检查和 LSP 集成。提供代码诊断、代码补全、悬停提示和风格分析。

- PEP8 风格分析
