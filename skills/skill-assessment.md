# Skill Assessment - 技能评估

## 技能描述

**Skill Assessment** 对 OpenClaw 技能进行轻量级静态分析，评估文档完整性、代码质量、配置友好性和维护信号。

- **最新版本**: 1.0.1
- **作者**: harrylabsj
- **更新时间**: 2026-04-21
- **标签**: assessment, analysis, quality

## 功能列表

- 文档完整性评估
- 代码质量分析
- 配置友好性检查
- 维护信号检测
- 静态分析报告

## 安装方式

```bash
clawhub install skill-assessment
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 适合 | 技能质量检查 |
| ECS 服务器 | ✅ 适合 | 适合自动化质量门禁 |

## 适用场景

- 技能质量审计
- 发布前检查
- 技能对比评估
