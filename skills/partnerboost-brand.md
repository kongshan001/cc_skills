# partnerboost-brand (PartnerBoost 商家 API)

## 技能描述

Call PartnerBoost merchant APIs to manage transactions, performance, billing, partners and more

调用 PartnerBoost 商家 API 管理交易、绩效、账单、合作伙伴等。

## 功能列表

- 交易管理
- 绩效查询
- 账单管理
- 合作伙伴管理
- 商家 API 集成

## 安装方式

```bash
clawhub install partnerboost-brand
```
