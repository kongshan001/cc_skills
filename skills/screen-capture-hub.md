# Screen Capture Hub

## 1. 背景需求

AI 代理在辅助用户时经常需要：
- 捕获屏幕截图进行分析
- 识别屏幕上的文字内容（OCR）
- 分析屏幕上的 UI 元素
- 对截图进行基本处理（裁剪、旋转、调整大小）

需要一个完整的屏幕捕获和处理工具包。

## 2. 目标

屏幕捕获和处理全功能支持：
- 整个屏幕或指定区域截图
- 截图保存到指定位置
- 截图文字识别（OCR，中英文支持）
- 基本图像处理（裁剪、旋转、调整大小）
- 屏幕内容分析

触发词：屏幕截图、截屏、查看屏幕、分析屏幕

## 3. 设计方案

**核心功能**：
| 功能 | 说明 |
|------|------|
| 全屏截图 | 使用 Python PIL |
| 区域截图 | 指定坐标范围 |
| OCR 识别 | 中英文文字提取 |
| 图像处理 | 裁剪、旋转、缩放 |
| 内容分析 | UI 元素识别 |

**技术栈**：Python PIL/Pillow（截图）、pytesseract（OCR）

## 4. 本地部署

```bash
clawhub install screen-capture-hub
```

**依赖**：
- Python 3
- PIL/Pillow
- pytesseract
- tesseract-ocr（系统包）

**安装额外依赖**：
```bash
pip install pillow pytesseract
# Linux
sudo apt install tesseract-ocr
# macOS
brew install tesseract
```

## 5. 效果展示

触发示例：
- "截取当前屏幕"
- "识别屏幕上的文字"
- "截取屏幕左上角区域"
- "分析这个截图中的内容"

执行效果：AI 执行截图、OCR 或分析操作。

## 6. 优缺点分析

| 优点 | 缺点 |
|------|------|
| 功能完整 | OCR 依赖外部引擎 |
| 中文支持好 | 精度依赖 tesseract |
| 简单易用 | 无视频捕获 |
| 适合 AI 分析 | 平台兼容性需验证 |

## 7. 平替对比

| 方案 | 特点 | 适用场景 |
|------|------|----------|
| Screen Capture Hub（当前） | 截图+OCR+处理 | AI 分析 |
| Playwright 截图 | Web 页面截图 | 浏览器内容 |
| 系统截图工具 | 本地快捷键 | 手动操作 |

## 8. 落地过程

**推荐安装评估**：
- **本地开发机** ⭐⭐⭐⭐⭐ — AI 视觉辅助必备
- **ECS 云服务器** ⭐⭐ — 无图形界面，不适用

**使用建议**：
1. 安装 tesseract-ocr 以支持 OCR
2. 定期清理截图文件
3. 中文 OCR 效果可能不如英文
