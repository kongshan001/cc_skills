# Mobile Development

## 技能描述

Mobile 是由 ivangdavila 开发的移动开发技能，专注于 iOS 和 Android 原生应用开发最佳实践，涵盖生命周期、权限、离线优先、性能、导航、通知等方面。

## 功能列表

### 生命周期管理
- 后台随时可能被杀死 - 后台前保存状态
- 恢复状态 - 用户期望继续之前的操作
- 处理低内存警告
- 后台任务时间限制

### 权限处理
- 上下文时请求权限 - 解释原因
- 拒绝时优雅降级
- 不请求不必要的权限
- 展示价值后重新请求

### 离线优先
- 假设网络不可靠
- 积极缓存
- 重试队列
- 冲突解决策略
- 显示同步状态

### 性能优化
- 60fps 目标
- 主线程仅用于 UI
- 内存优化
- 电池意识
- 启动时间 < 2 秒

### 导航模式
- 遵循平台约定 (iOS 手势、Android 返回)
- 导航栈可管理
- 深度链接到任何屏幕
- 返回时保留滚动位置

### 通知
- 推送 - 时间敏感外部事件
- 本地 - 提醒、定时器
- 质量 > 数量
- 可操作通知

### 深度链接
- Universal Links (iOS) / App Links (Android)
- 应用未安装时优雅降级
- 安全解析参数

### 输入处理
- 键盘避免
- 适当关闭键盘
- 输入附件
- 粘贴、自动填充支持

### 触摸和手势
- 44pt 最小触摸目标
- 不覆盖系统手势
- 手势可发现性
- 触觉反馈

### 无障碍
- VoiceOver / TalkBack 测试
- 动态类型支持
- 足够对比度
- 所有交互元素标签

### 测试
- 真机测试必备
- 多个 OS 版本
- 不同屏幕尺寸
- 网络条件测试

## 安装方式

```bash
# 使用 ClawHub 安装
clawhub install mobile
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 推荐 | 移动应用开发必备 |
| ECS/服务器 | ❌ 不推荐 | 服务端无需此技能 |

### 本地部署
- iOS/Android 应用开发
- 跨平台框架 (React Native, Flutter)
- 移动端性能优化

### ECS 部署
- 不适用（客户端技能）
- 后端 API 开发可配合 FastAPI
