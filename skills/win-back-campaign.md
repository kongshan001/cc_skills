# Win-Back Campaign — 客户召回营销

## 背景需求

电商卖家需要召回沉睡客户，通过个性化重新参与序列提升复购。

## 目标

设计针对沉睡客户的自动召回营销活动，跨email、SMS、付费广告的个性化序列。

## 设计方案

- 基于RFM的沉睡客户识别
- 个性化邮件序列
- SMS/广告召回

## 本地部署

```bash
npm i -g clawhub
clawhub install win-back-campaign
```

## 推荐安装

**本地 / ECS**：✅ 推荐 - 定时任务需持续运行

## 效果展示

- 召回漏斗
- 自动化序列

## 优缺点分析

**优点**：
- 多渠道覆盖
- RFM模型

**缺点**：
- 需配合数据源
