# Apple Developer Toolkit

## 技能描述

一体化 Apple 开发者技能，包含三个集成工具：(1) 跨 Apple 框架、符号、API 的文档搜索，(2) VisionPro 空间计算开发，(3) Swift 调试辅助。

## 功能列表

- Apple 文档搜索
- VisionPro 开发
- Swift 调试
- 框架 API 查询
- 符号搜索

## 安装方式

```bash
clawhub install apple-developer-toolkit
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐⭐ | Apple 开发必备 |
| ECS | ⭐⭐⭐ | Mac 绑定开发 |

## 适用场景

- Apple 平台开发
- VisionOS 应用开发
- Swift 问题调试
