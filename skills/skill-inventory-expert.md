# Skill Inventory Expert - 技能清单专家

## 技能描述

**Skill Inventory Expert** 是 AI Agent 技能清单和能力评估指南。帮助 Agent 发现、编目并以结构化 JSON 格式报告已安装的技能。

- **最新版本**: 1.0.0
- **作者**: saqierma-cyber
- **更新时间**: 2026-03-18
- **标签**: inventory, assessment, catalog

## 功能列表

- 发现已安装技能
- 技能编目
- JSON 格式报告
- 能力评估

## 安装方式

```bash
clawhub install skill-inventory-expert
```

| 场景 | 推荐 | 说明 |
|------|------|------|
