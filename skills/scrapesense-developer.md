## 技能描述

Comprehensive ScrapeSense public API developer skill for scan orchestration, places extraction, campaign lifecycle, email cleaning, billing endpoints, and API integration.

全面的 ScrapeSense 公共 API 开发技能，支持扫描编排、地点提取、活动生命周期、邮件清理、计费端点和 API 集成。
