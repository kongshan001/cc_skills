## 技能描述

Build or improve Telegram inline-button file browser. Allows users to browse, search, and manage files shared in Telegram channels and groups.

构建或改进 Telegram 内联按钮文件浏览器。允许用户浏览、搜索和管理在 Telegram 频道和群组中分享的文件。

## 功能列表

- Telegram 文件浏览
- 内联按钮交互
- 文件搜索功能
- 文件管理
- 频道和群组文件访问
- 批量下载和操作
