# 龙虾文游系统 - Text Adventure Game Skill

## 技能描述

智能文字游戏生成系统，支持自定义剧本，自由互动剧情，精美文字排版

## 功能列表

- 自定义剧本生成
- 自由互动剧情
- 精美文字排版
- 文字冒险体验

## 安装方式

```bash
clawhub install text-adventure-game-skill
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 文字冒险体验佳 |
| ECS服务器 | ⭐⭐⭐⭐⭐ | 适合故事爱好者 |

**推荐：本地/ECS均可**

- 中文支持好
- 适合角色扮演爱好者
- 文字排版精美
