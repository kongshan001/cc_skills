## 技能描述

Self-reflection + Self-criticism + Self-learning + Self-organizing memory. Agent evaluates its own work, catches mistakes, and improves permanently.

自我反思 + 自我批评 + 自我学习 + 自我组织记忆。Agent 评估自己的工作，发现错误，并永久改进。

## 功能列表

- 自我反思 (Self-reflection)
- 自我批评 (Self-criticism)
- 自我学习 (Self-learning)
- 自我组织记忆 (Self-organizing memory)
- 错误捕捉
- 永久改进

## 安装方式

```bash
# 使用 ClawHub 安装
npx clawhub@latest install self-improving

# 或全局安装 ClawHub 后安装
npm i -g clawhub
clawhub install self-improving
```

## 推荐安装评估

### 本地安装 ⭐⭐⭐⭐⭐
- 适合场景：日常任务处理、复杂项目开发
- 优势：持续学习、积累经验
- 要求：Node.js 环境

### ECS 安装 ⭐⭐⭐⭐⭐
- 适合场景：长期运行任务、团队协作
- 优势：持久化学习成果、团队共享
- 要求：ECS 实例 + 持久存储

## 优缺点分析

### 优点
- 持续自我改进
- 记忆和组织知识
- 减少重复错误

### 缺点
- 需要持久化存储
- 可能产生大量学习数据
