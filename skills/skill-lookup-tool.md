# Skill Lookup - 技能查找工具

## 技能描述

**Skill Lookup** 从 prompts.chat registry 搜索、检索和安装 Agent Skills。使用 MCP 工具实现。

- **最新版本**: 1.0.0
- **作者**: zhengxinjipai
- **更新时间**: 2026-04-21
- **标签**: search, mcp, registry

## 功能列表

- 从 registry 搜索技能
- 技能检索
- MCP 安装集成
- 技能目录浏览

## 安装方式

```bash
clawhub install skill-lookup-tool
```
