## 技能描述

Inspect containers, logs, and images via podman.

通过 podman 检查容器、日志和镜像。

## 功能列表

- 容器检查
- 日志查看
- 镜像管理
- Podman 集成

## 安装方式

```bash
clawhub install docker-ctl
```
