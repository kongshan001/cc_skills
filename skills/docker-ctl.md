# Docker Ctl

| 项目 | 内容 |
|------|------|
| **技能名称** | docker-ctl |
| **所有者** | xejrax |
| **版本** | 1.0.0 |
| **更新时间** | 2026-02-28 |

## 技能描述

通过 podman 检查容器、日志和镜像。

## 功能列表

- 容器检查
- 日志查看
- 镜像管理

## 安装方式

```bash
clawhub install docker-ctl
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| **本地开发** | ⭐⭐⭐⭐ | 容器管理 |
| **ECS服务器** | ⭐⭐⭐⭐ | 运维监控 |

## 使用场景

- Podman 替代 Docker
- 轻量容器管理
