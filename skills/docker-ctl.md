# Docker Ctl

> 技能Slug: docker-ctl  
> 版本: 1.0.0  
> 作者: xejrax  
> 更新时间: 2026-02-28

## 技能描述

Inspect containers, logs, and images via podman

通过 podman 检查容器、日志和镜像。

## 功能列表

- 容器检查
- 日志查看
- 镜像管理
- podman 集成

## 安装方式

```bash
clawhub install docker-ctl
```

## 推荐安装评估

### 本地开发环境 ✅ 推荐

- 替代 Docker 的轻量选择
- 适合 macOS/Linux

### ECS/服务器 ✅ 推荐

- 无守护进程容器运行
- 资源占用更低
