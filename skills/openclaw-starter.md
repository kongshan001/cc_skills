# openclaw-starter (OpenClaw 新手引导)

## 技能描述

OpenClaw 新手一站式引导。环境检测、安装状态检查、Gateway 配置、必备 Skills 推荐、渠道配置、首次使用教程。5 分钟快速上手 OpenClaw。

## 功能列表

- 环境检测
- 安装状态检查
- Gateway 配置
- 必备 Skills 推荐
- 渠道配置
- 首次使用教程
- 配置文件模板

## 安装方式

```bash
clawhub install openclaw-starter
```
