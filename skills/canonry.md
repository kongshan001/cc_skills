# Canonry — Agent-First AEO 运行平台

## 背景需求

AEO (Agentic Engine Optimization) 是新兴的 AI 搜索引擎优化方向，需要专门的工具平台来管理和优化 AI Agent 的输出表现。随着更多 AI 搜索引擎（如 Perplexity、Arc Search）崛起，网站和内容开发者需要针对这些 AI 代理进行优化。

## 目标

提供 Agent-first 的 AEO 操作平台，帮助用户优化其在 AI 搜索引擎中的可见度和排名。

## 设计方案

- 专门针对 AI Agent 的 SEO 优化工具
- 提供性能监控和数据洞察
- 支持多 AI 搜索引擎优化

## 本地部署

```bash
npm i -g clawhub
clawhub install canonry
```

## 效果展示

- AEO 性能仪表板
- 关键词追踪
- 排名分析

## 优缺点分析

**优点**：
- 专注于新兴 AEO 领域
- 平台化解决方案

**缺点**：
- 领域较新，生态待完善
