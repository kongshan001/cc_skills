# Web Browsing

## 技能概述

**Slug**: `web-browsing`  
**作者**: tankebuaa  
**版本**: 1.0.0  
**更新日期**: 2026-03-14

## 背景需求

为 AI 提供完整的网页浏览和交互能力。

## 功能列表

1. 浏览器自动化控制
2. 网页内容抓取
3. 表单填写与提交
4. 页面交互操作

## 安装方式

```bash
clawhub install web-browsing
```

## 推荐安装评估

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 本地开发 | ⭐⭐⭐⭐ | 需配合浏览器环境使用 |
| ECS 服务器 | ⭐⭐⭐⭐ | 支持无头浏览器模式 |

## 优缺点分析

**优点**:
- 功能全面的浏览器自动化
- 支持复杂网页交互

**缺点**:
- 需要浏览器环境
- 可能被反爬虫检测
