# Kuuila互动游戏框架

## 技能描述

通用互动游戏框架，支持单人文字冒险、猜谜及多人武侠游戏，含私聊控制群游戏和实时状态同步。

Tags: adventure, game, interactive

## 功能列表

- 单人文字冒险
- 猜谜游戏
- 多人武侠游戏
- 私聊控制群游戏
- 实时状态同步

## 安装方式

```bash
clawhub install kuuila-game
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 完整游戏框架 |
| ECS服务器 | ⭐⭐⭐⭐⭐ | 多人游戏需要服务器 |

**推荐：本地/ECS均可**

- 框架完整，支持多种玩法
- 武侠题材有特色
- 实时同步适合多人
