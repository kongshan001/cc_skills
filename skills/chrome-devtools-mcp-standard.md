# Chrome DevTools MCP (Standard)

## 技能描述

官方 Google 项目，通过 Chrome DevTools MCP 为 agents 提供完整的 Chrome DevTools 访问。29 个工具支持输入自动化、截图、console 读取等。

## 功能列表

- 浏览器自动化
- 截图 capture
- Console 读取
- 网络监控
- 元素操作
- 29+ DevTools 工具

## 安装方式

```bash
clawhub install chrome-devtools-mcp-standard
```
