# Docker Compose

## 技能概述

**Slug**: `docker-compose`  
**作者**: ivangdavila  
**版本**: 1.0.0  
**更新日期**: 2026-02-26

## 背景需求

Define multi-container applications with proper dependency handling, networking, and volume management.

## 功能列表

1. 多容器应用定义
2. 依赖处理
3. 网络配置
4. 卷管理

## 安装方式

```bash
clawhub install docker-compose
```

## 推荐安装评估

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 多容器编排必备 |
| ECS 服务器 | ⭐⭐⭐⭐⭐ | 部署复杂应用 |

## 优缺点分析

**优点**:
- 完整的编排能力
- 依赖管理完善

**缺点**:
- 需要 Docker Compose 环境
