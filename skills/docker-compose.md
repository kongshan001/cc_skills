# Docker Compose

- **Slug**: docker-compose
- **作者**: ivangdavila
- **版本**: 1.0.0
- **更新时间**: 2026-02-26
- **标签**: docker, compose, multi-container

## 功能列表

1. 多容器应用定义
2. 依赖关系处理
3. 网络配置管理
4. 卷挂载管理
5. 服务编排最佳实践

## 安装方式

```bash
clawhub install docker-compose
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 本地多服务开发必备 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 微服务部署必备 |

## 适用场景

- 本地开发环境搭建
- 微服务架构部署
- 开发/测试环境管理
- CI/CD 环境配置

## 优缺点分析

**优点**:
- 简化多容器管理
- 支持依赖声明
- 环境一致性保证

**缺点**:
- 不适合大规模生产环境（推荐 Kubernetes）
- 集群能力有限
