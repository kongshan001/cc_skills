# Docker Compose

- **Slug**: docker-compose
- **作者**: ivanggavila
- **版本**: 1.0.0
- **更新时间**: 2026-02-26

## 背景需求

多容器应用需要定义依赖关系、网络和卷管理。

## 目标

提供 Docker Compose 最佳实践和工作流。

## 功能列表

- 多容器应用定义
- 依赖处理
- 网络配置
- 卷管理
- 服务编排

## 安装方式

```bash
clawhub install docker-compose
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 微服务开发必备
- **ECS/服务器**: ⭐⭐⭐⭐⭐ 容器编排必备
