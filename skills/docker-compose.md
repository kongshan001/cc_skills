# Docker Compose

> Define multi-container applications with proper dependency handling, networking, and volume management.

## 技能概述

Docker Compose 技能提供多容器应用编排能力，处理服务依赖、网络和卷管理。

## 功能列表

- 多容器应用定义
- 服务依赖管理
- 网络配置
- 卷管理
- 环境变量配置

## 安装方式

```bash
clawhub install docker-compose
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | 本地多容器开发必需 |
| ECS | ⭐⭐⭐⭐⭐ | 微服务部署必备 |
