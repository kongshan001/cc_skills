# Vibe Coding Skill

> Vibe Coding 技能

## 技能描述

使用 Vibe Coding 5 阶段工作流（需求、架构、代码生成、调试、迭代）指导 AI 辅助开发。

## 核心功能

1. **需求阶段** - Requirements
2. **架构阶段** - Architecture
3. **代码生成** - Code generation
4. **调试阶段** - Debugging
5. **迭代阶段** - Iteration

## 安装方式

```bash
clawhub install vibe-coding-skill
```
