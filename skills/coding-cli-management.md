# Coding CLI Management

## 技能描述

代表 Workers 管理和执行 AI 编码 CLI 工具，在 Worker 工作空间中运行编码提示并返回结果。

## 功能列表

- CLI 工具管理
- 编码任务执行
- 工作空间集成
- 结果返回

## 安装方式

```bash
clawhub install coding-cli-management
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐ | CLI 开发 |
| ECS | ⭐⭐⭐⭐ | 远程执行 |

## 适用场景

- 批量编码任务
- CLI 工具封装
- 自动化编码
