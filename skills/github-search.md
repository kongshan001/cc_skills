# GitHub Search

## 技能概述

- **Slug**: github-search
- **名称**: GitHub Search
- **作者**: linshengli
- **版本**: 1.0.0
- **更新时间**: 2026-04-09

## 背景需求

技术调研需要搜索 GitHub 仓库来了解最新开源项目，按关键词、语言、stars、更新时间筛选。

## 目标

GitHub 仓库深度搜索与分析。支持按关键词、语言、stars、更新时间筛选，获取细分领域最新开源项目。专为技术调研设计。

## 功能列表

- 按关键词搜索仓库
- 按语言筛选
- 按 stars 数量筛选
- 按更新时间筛选
- 细分领域调研
- 最新开源项目发现

## 安装方式

```bash
clawhub install github-search
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS 服务器**: ⭐⭐⭐ 推荐

**理由**: 技术调研和了解行业趋势的利器，可以快速发现相关领域的开源项目。

## 优缺点分析

### 优点

1. 深度 GitHub 搜索
2. 多维度筛选
3. 适合技术调研
4. 发现最新项目

### 缺点

1. 依赖 GitHub API
2. 可能受 rate limit 影响
