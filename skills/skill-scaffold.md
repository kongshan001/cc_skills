# Skill Scaffold - 技能脚手架

## 技能描述

**Skill Scaffold** 是一个AI Agent技能脚手架CLI工具。可以为 OpenClaw、Moltbot、Claude、Cursor、ChatGPT、Copilot 即时创建技能。支持 vibe-coding，MCP 兼容。

- **最新版本**: 1.0.4
- **作者**: nextfrontierbuilds
- **更新时间**: 2026-04-21
- **标签**: ai, clawdbot, cli, generator, scaffold, skill, openclaw, moltbot, claude, cursor, copilot, mcp

## 功能列表

- 快速创建 OpenClaw 技能
- 支持创建 Claude/Cursor/ChatGPT/Copilot 技能
- MCP 兼容
- Vibe-coding 友好
- CLI 交互式生成

## 安装方式

```bash
clawhub install skill-scaffold
```
