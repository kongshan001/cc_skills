# Context Driven Development

## 技能描述

将项目上下文视为与代码同等的可管理工件。通过结构化的上下文文档（product.md、tech-stack.md、workflow.md）实现一致的AI交互和团队协作。是使用AI辅助开发项目的必备技能。

## 核心功能

- 项目上下文文档管理（product.md、tech-stack.md、workflow.md）
- 结构化的AI交互模式
- 团队上下文同步
- 项目规范一致性维护

## 安装方式

```bash
clawhub install context-driven-development
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐⭐ 适用

**理由**: 该技能专注于本地开发工作流和团队协作，在本地环境效果最佳。适合任何规模的开发团队，特别是使用AI辅助开发的团队。
