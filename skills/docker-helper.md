# Docker Helper

- **Slug**: docker-helper
- **作者**: ckchzh
- **版本**: 2.0.0
- **更新时间**: 2026-03-17

## 背景需求

Docker 使用中需要快速生成 Dockerfile、编排文件、调试排错。

## 目标

提供 Docker 全方位辅助工具。

## 功能列表

- Dockerfile 生成
- docker-compose 编排
- 命令速查
- 调试排错
- 镜像优化
- 仓库配置

## 安装方式

```bash
clawhub install docker-helper
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 开发效率提升
- **ECS/服务器**: ⭐⭐⭐⭐ 运维辅助
