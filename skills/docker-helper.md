# Docker Helper

> Dockerfile生成、docker-compose编排、命令速查、调试排错、镜像优化、仓库配置

## 技能概述

Docker Helper 提供完整的中文 Docker 支持，包括 Dockerfile 生成、Compose 编排和调试排错。

## 功能列表

- Dockerfile 生成
- docker-compose 编排
- 命令速查
- 调试排错
- 镜像优化
- 仓库配置

## 安装方式

```bash
clawhub install docker-helper
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | 中文界面友好 |
| ECS | ⭐⭐⭐⭐⭐ | 服务器运维必备 |

## 优缺点

- ✅ 中文支持
- ✅ 功能全面
