# Webperf

> Web 性能测量与调试

## 概述

**Slug:** webperf  
**Owner:** nucliweb  
**版本:** 0.1.0  
**更新时间:** 2026-04-08

## 功能列表
- Web 性能测量
- 性能审计
- 调试工具
- 性能分析

## 使用场景
当用户询问网页性能、需要审计页面或说"analyze performance"、"debug performance"时使用。

## 安装方式
```bash
clawhub install webperf
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 推荐 | 性能优化工具 |
| ECS 服务器 | ✅ 推荐 | 服务器性能监控 |
