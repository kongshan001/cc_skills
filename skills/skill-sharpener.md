# Skill Sharpener - 技能磨刀石

## 技能描述

**Skill Sharpener** 对现有 Agent Skill (SKILL.md) 进行全面质量评估并生成优化建议报告。检查技能是否符合最佳实践、描述是否有效触发、结构是否合理、内容是否简洁。

- **最新版本**: 1.0.0
- **作者**: liuxxxu
- **更新时间**: 2026-03-10
- **标签**: assessment, optimization, quality

## 功能列表

- 技能质量评估
- 最佳实践检查
- 结构合理性分析
- 优化建议生成
- 内容简洁性审查

## 安装方式

```bash
clawhub install skill-sharpener
```
