# Skill Inventory - 技能目录管理器

## 技能描述

**Skill Inventory** 是技能目录管理器，扫描已安装的技能，生成/更新技能目录文件。

- **最新版本**: 1.0.0
- **作者**: bohell
- **更新时间**: 2026-03-13
- **标签**: inventory, catalog, management

## 功能列表

- 扫描已安装技能
- 生成技能目录文件
- 更新技能清单
- 列出系统/用户级技能

## 安装方式

```bash
clawhub install skill-inventory
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 适合 | 技能清单管理 |
| ECS 服务器 | ✅ 适合 | 资产统计 |

## 适用场景

- 查看技能列表
- 更新技能目录
- 资产统计
