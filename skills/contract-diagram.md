# Contract Diagram

> 图表即契约

## 技能描述

用图表作为 AI 开发的契约，明确需求和实现之间的约定。

## 核心功能

1. **图表驱动开发** - 用图表定义需求
2. **视觉契约** - 可视化的开发契约
3. **需求对齐** - 团队需求对齐

## 安装方式

```bash
clawhub install contract-diagram
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐ | 适合视觉化需求 |
