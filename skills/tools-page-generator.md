# Tools Page Generator

## 技能描述

当用户想要创建、优化或审计免费工具页面时使用。也用于用户提及 "free tools"、"tools page"、"toolkit" 等。

## 功能列表

- 工具页面创建
- 页面优化
- 审计功能
- 免费工具聚合

## 安装方式

```bash
clawhub install tools-page-generator
```

|------|--------|------|
