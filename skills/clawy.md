# clawy (AI Agent 头像生成)

## 技能描述

Bring your agent to life. Generate stable agent avatars and short image-driven adventure arcs. Use when creating or refining a mascot/agent identity, preserving...

为你的 AI Agent 生成稳定的头像和短篇图像驱动的冒险故事。用于创建或优化助手/Agent 身份形象。

## 功能列表

- AI Agent 头像生成
- 稳定的角色形象创建
- 图像驱动的冒险故事生成
- Agent 形象定制
- 官方静态 PNG 回退支持
