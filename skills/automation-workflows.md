# Automation Workflows 技能调研报告

## 1. 背景需求

作为独立创业者（solopreneur），需要自动化重复性任务来节省时间并扩展运营规模。手动处理日常事务效率低下，需要系统化的自动化工作流。

## 2. 目标

帮助用户识别、设计和实现自动化工作流，包括：
- 识别可自动化的重复任务
- 跨工具构建工作流
- 设置触发器和动作
- 优化现有自动化

## 3. 设计方案

### 核心能力
- **机会识别**: 分析业务流程，找出可自动化点
- **工作流设计**: 使用图表和流程图设计自动化
- **工具选择**: 推荐合适工具（Zapier、Make、n8n）
- **测试与维护**: 确保自动化稳定运行

### 支持平台
- Zapier
- Make (原 Integromat)
- n8n

### 触发关键词
`automate`, `automation`, `workflow automation`, `save time`, `reduce manual work`, `automate my business`, `no-code automation`

## 4. 本地部署

```bash
# 安装 Automation Workflows 技能
clawhub install automation-workflows
```

## 5. 效果展示

典型应用场景：
- 表单提交自动通知
- 数据同步到电子表格
- 定时报告生成
- 社交媒体自动发布
- 客户消息自动回复

## 6. 优缺点分析

### 优点
- 专注于 solopreneur 场景，针对性强
- 覆盖主流无代码自动化平台
- 包含完整的从识别到维护流程

### 缺点
- 需要配合第三方平台使用
- 高级复杂场景支持有限

## 7. 平替对比

| 技能 | 特点 |
|------|------|
| automation-workflows | 无代码工作流设计 |
| ai-web-automation | 浏览器自动化（Web 任务） |
| agentic-workflow-automation | 代理驱动的自动化 |

## 8. 落地过程

1. 执行 `clawhub install automation-workflows`
2. 描述需要自动化的业务场景
3. 根据技能指导识别自动化机会
4. 选择合适工具并设计工作流
