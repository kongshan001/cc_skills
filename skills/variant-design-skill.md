## 技能描述

Generates 3 distinct, high-quality UI design variants based on user requirements, with detailed explanations of design choices.

根据用户需求生成 3 个独特、高质量的 UI 设计变体，并提供设计选择的详细解释。

## 功能列表

- UI 设计变体生成
- 生成 3 个不同设计方案
- 高质量设计输出
- 设计选择解释
- 基于用户需求定制
