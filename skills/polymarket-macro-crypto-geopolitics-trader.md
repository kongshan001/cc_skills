# Polymarket Macro Crypto Geopolitics — 地缘政治与加密联动交易

## 背景需求

地缘政治事件与加密市场存在联动，需要捕捉这种滞后关系获取alpha。

## 目标

交易地缘政治升级市场与加密价格阈值市场之间的滞后效应。

## 设计方案

- 地缘政治事件监控
- 加密价格联动分析
- 滞后套利

## 本地部署

```bash
npm i -g clawhub
clawhub install polymarket-macro-crypto-geopolitics-trader
```
