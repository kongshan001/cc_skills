# x-openclaw (X/Twitter 帖子提取)

## 技能描述

Extract posts from X (Twitter) timeline or profile pages with engagement metrics, media URLs, and download local copies. Use when asked to scrape, list, or a...

从 X (Twitter) 时间线或个人资料页面提取帖子，包含互动指标、媒体 URL，并下载本地副本。

## 功能列表

- X/Twitter 时间线提取
- 个人资料页面爬取
- 互动指标提取
- 媒体 URL 提取
- 本地文件下载
- 帖子元数据保存
- 严格输出格式

## 安装方式

```bash
clawhub install x-openclaw
```
