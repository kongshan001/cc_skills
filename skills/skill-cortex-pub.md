# Skill Cortex Pub - 技能皮层发布

## 技能描述

**Skill Cortex Pub** 是系统的能力皮层。当缺乏能力时，它自主从 ClawHub 或 GitHub 获取 Skills，使用后释放它们。

- **最新版本**: 0.0.2
- **作者**: ankwu001
- **更新时间**: 2026-02-27
- **标签**: cortex, autonomous, capability

## 功能列表

- 自主技能获取
- 按需加载技能
- 使用后释放资源
- 动态能力扩展

## 安装方式

```bash
clawhub install skill-cortex-pub
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ⚠️ 谨慎 | 较新版本，需观察 |
| ECS 服务器 | ⚠️ 谨慎 | 生产环境需测试 |

## 适用场景

- 动态能力扩展
- 按需技能加载
- 资源优化管理
