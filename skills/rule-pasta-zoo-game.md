# 动物园规则怪谈 (rule-pasta-zoo-game)

## 背景需求

规则怪谈是中文网络流行的叙事类型，通过看似合理但隐含矛盾的规则构建恐怖氛围。创作者需要系统化创作工具。

## 目标

提供规则怪谈创作系统，支持生成符合该类型特征的恐怖叙事内容。

## 设计方案

- **核心特征**:
  - 看似合理但隐含矛盾的规则
  - 公告体/第一人称叙事
  - 恐怖氛围营造
  - 超自然暗示
- **触发方式**: 关键词触发

## 本地部署

```bash
clawhub install rule-pasta-zoo-game
```

## 效果展示

- 规则怪谈内容生成
- 恐怖氛围渲染
- 多种叙事体裁

## 优缺点分析

✅ 优点:
- 细分领域垂直
- 叙事风格独特
- 创作效率高

❌ 缺点:
- 主题小众
- 恐怖内容可能不适合所有人

## 落地过程

1. 触发创作
2. 描述场景或角色
3. 生成规则怪谈内容
