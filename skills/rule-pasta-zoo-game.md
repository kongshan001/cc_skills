# 动物园规则怪谈 - Rule Pasta Zoo Game

## 技能描述

规则怪谈是一种近年来在中文网络文学与亚文化语境中兴起的叙事类型或恐怖文学子类。

其核心特征在于：通过一系列看似合理、逻辑自洽但实则隐含矛盾、悖论或超自然暗示的"规则"来构建恐怖氛围，并引导读者在解读规则的过程中逐步感知异常与不安。

## 功能列表

- 规则怪谈叙事生成
- 恐怖氛围构建
- 逻辑悖论设计
- 第一人称/公告体文本
