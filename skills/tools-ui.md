# Tools Ui

## 技能描述

React/Next.js 的工具生命周期 UI 组件，来自 ui.inference.sh。显示工具调用状态：pending、progress、approval required、results。

## 功能列表

- 工具调用状态显示
- Pending 状态指示
- 进度条显示
- 审批请求 UI
- 结果展示组件

## 安装方式

```bash
clawhub install tools-ui
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐ | 前端项目开发 |
| ECS | ⭐⭐ | 需要前端环境 |

## 适用场景

- 构建 AI 工具调用界面
- 实现工具审批流程
- 展示工具执行状态
