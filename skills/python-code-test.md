# Python Code Tester

## 技能概述

- **Slug**: python-code-test
- **名称**: Python Code Tester
- **作者**: zhouhh2017
- **版本**: 1.0.3
- **更新时间**: 2026-04-09

## 背景需求

编写测试用例是保证代码质量的重要环节，但手动编写测试耗时且容易遗漏边界情况。自动化测试生成可以大大提高效率。

## 目标

代码功能测试 skill，根据用户需求搜索代码、生成测试用例、执行测试并修复问题。

## 功能列表

- 根据需求搜索代码
- 自动生成测试用例
- 执行测试
- 自动修复测试问题
- 测试覆盖率分析

## 安装方式

```bash
clawhub install python-code-test
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS 服务器**: ⭐⭐⭐⭐ 推荐

**理由**: 测试是软件质量的重要保障，此技能可以自动化生成测试用例，大大减少测试编写时间。

## 优缺点分析

### 优点

1. 自动化测试生成
2. 减少手工测试工作量
3. 提高测试覆盖率
4. 自动修复测试问题

### 缺点

1. 可能生成不完整的测试
2. 需要人工审核测试质量
