# Web Learner (自主上网学习技能)

## 技能概述

**Slug**: `web-learner`  
**作者**: jackeven02  
**版本**: 1.0.0  
**更新日期**: 2026-02-25

## 背景需求

让 AI 能够主动搜索、浏览和从互联网获取知识。当用户要求了解最新信息、学习新知识、查询新闻、获取某个主题的详细信息，或需要从网络上获取数据时触发。

## 功能列表

1. 主动上网搜索
2. 浏览网页获取知识
3. 新闻查询
4. 主题详细信息获取
5. 网络数据抓取

## 安装方式

```bash
clawhub install web-learner
```

## 推荐安装评估

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 轻量级搜索能力 |
| ECS 服务器 | ⭐⭐⭐⭐⭐ | 无需额外依赖 |

## 优缺点分析

**优点**:
- 主动学习能力强
- 适用于知识问答场景

**缺点**:
- 依赖外部搜索 API
