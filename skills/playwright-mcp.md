# Playwright MCP

> 技能Slug: playwright-mcp  
> 版本: 1.0.0  
> 作者: spiceman161  
> 更新时间: 2026-02-26

## 技能描述

Browser automation via Playwright MCP server. Navigate websites, click elements, fill forms, extract data, take screenshots, and perform full browser automation workflows.

通过 Playwright MCP 服务器实现浏览器自动化。

## 功能列表

- 网页导航
- 元素点击
- 表单填写
- 数据提取
- 截图
- 完整浏览器自动化工作流

## 安装方式

```bash
clawhub install playwright-mcp
```

## 推荐安装评估

### 本地开发环境 ✅ 推荐

- 需要 Node.js 环境
- 适合 E2E 测试

### ECS/服务器 ✅ 推荐

- 自动化测试流水线
- 网页数据抓取
