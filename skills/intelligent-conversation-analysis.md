# Intelligent Conversation Analysis - 智能对话分析

## 基本信息

- **技能名称**: intelligent-conversation-analysis
- **版本**: 1.0.0
- **作者**: 未知
- **创建时间**: 2026-03-15
- **更新时间**: 2026-03-15

## 技能描述

该技能调用阿里云晓蜜 CCAI AIO，对客服通话或文字对话进行 AI 智能分析，支持摘要生成、服务质量评估等功能。

使用阿里云晓蜜 CCAI AIO 服务，对客服通话或文字对话进行 AI 智能分析。

## 功能列表

- 客服通话分析
- 文字对话分析
- 摘要生成
- 服务质量评估
- 阿里云晓蜜集成

## 安装方式

```bash
clawhub install intelligent-conversation-analysis
```
