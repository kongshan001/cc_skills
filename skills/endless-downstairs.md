# Endless Downstairs - 文字冒险游戏

## 技能描述

Interactive interface for the text adventure game Endless Downstairs, providing game state management and event selection handling.

用于用户体验验文字冒险游戏 Endless Downstairs。

## 功能列表

- 游戏状态管理
- 事件选择处理
- 文字冒险交互

## 安装方式

```bash
clawhub install endless-downstairs
```
