## 技能描述

Create AI images with GPT Image, Gemini Nano Banana, FLUX, Imagen, and top providers using prompt engineering, style control, and smart editing.

使用 GPT Image、FLUX、Imagen 等顶级提供商创建 AI 图像，支持提示词工程、风格控制和智能编辑。

## 功能列表

- GPT Image 生成
- FLUX 生成
- Imagen 生成
- Gemini Nano Banana 支持
- 提示词工程
- 风格控制
- 智能编辑

## 安装方式

```bash
# 使用 ClawHub 安装
npx clawhub@latest install image-generation

# 或全局安装 ClawHub 后安装
npm i -g clawhub
clawhub install image-generation
```

## 推荐安装评估

### 本地安装 ⭐⭐⭐⭐
- 适合场景：本地创意设计、图像生成
- 优势：保护隐私、无需上传
- 要求：Node.js 环境

### ECS 安装 ⭐⭐⭐⭐⭐
- 适合场景：批量图像生成服务
- 优势：可构建图像生成 API
- 要求：ECS 实例

## 优缺点分析

### 优点
- 多种 AI 图像提供商
- 强大的风格控制
- 支持编辑功能

### 缺点
- 需要各平台 API
- 可能产生费用
