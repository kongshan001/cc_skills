# 即时游戏设计引擎 (game)

## 背景需求

游戏设计师希望快速验证游戏概念，传统开发流程过于繁琐。需要一种方式：用一句话描述游戏想法，立即获得完整可玩的设计。

## 目标

将自然语言游戏描述转换为完整的可玩游戏设计，支持桌游、卡牌、派对游戏等多种类型。

## 设计方案

- **输入**: 一句话游戏概念描述
- **输出**: 完整游戏设计文档和可玩版本
- **支持类型**: 桌游、卡牌、派对游戏、策略、模拟等

## 本地部署

```bash
clawhub install game
```

## 效果展示

- 一句话生成游戏
- 多种游戏类型支持
- 即时可玩

## 优缺点分析

✅ 优点:
- 快速原型验证
- 覆盖类型广
- 零开发门槛

❌ 缺点:
- 生成质量依赖描述详细程度
- 复杂规则可能简化

## 落地过程

1. 描述游戏想法
2. AI 生成完整设计
3. 开始游戏
