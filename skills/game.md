# Game - 即时游戏设计引擎

## 技能描述

The instant game design engine for AI agents. Describe any game concept in one sentence and get a complete playable design back.

支持：Board games, card games, party games, strategy games, simulation 等多种类型。

Tags: design, game, gamification, incentive, party, play, simulation, strategy, theory

## 功能列表

- 一句话游戏概念生成完整可玩设计
- 桌游/卡牌游戏/派对游戏设计
- 策略游戏/模拟游戏生成
- 游戏化机制设计

## 安装方式

```bash
clawhub install game
```
