# Tianyi Cloud Game - 天翼云游戏

## 技能描述

天翼云游戏 - 一键打开天翼云游戏H5平台，畅玩云端游戏

## 功能列表

- 天翼云游戏H5平台快速访问
- 云端游戏体验

## 安装方式

```bash
clawhub install tianyi-cloud-game
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐ | 需浏览器访问外部服务 |
| ECS服务器 | ⭐⭐⭐ | 引导跳转技能 |

**推荐：本地开发**

- 简单跳转技能，功能单一
- 适合快速访问云游戏平台
