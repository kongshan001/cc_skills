# session-memory-enhanced (会话记忆增强)

## 技能描述

Session-Memory Enhanced v4.0 - 统一增强版。融合 session-memory + memu-engine 核心功能。特性：结构化提取 + 向量检索 + 不可变分片 + 三位一体自动化 + 多代理隔离 + AI 摘要 + 零配置启动。

## 功能列表

- 结构化信息提取
- 向量检索
- 不可变分片存储
- 多代理隔离
- AI 摘要生成
- 零配置启动
- 临时配置管理
- 多种运行方式支持
