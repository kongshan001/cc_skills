# Skill Test - 技能测试

## 技能描述

**Skill Test** 用于在使用或发布技能前进行测试。在隔离环境中试用、比较、评估技能，不影响现有环境。

- **最新版本**: 1.0.0
- **作者**: ivangdavila
- **更新时间**: 2026-02-25
- **标签**: testing, evaluation, sandbox

## 功能列表

- 技能隔离测试
- 多技能对比评估
- 发布前验证
- 环境无污染测试

## 安装方式

```bash
clawhub install skill-test
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 强烈推荐 | 调试技能必备 |
| ECS 服务器 | ✅ 适合 | CI/CD 流程集成 |

## 适用场景

- 发布前技能验证
- 多技能对比测试
- 隔离环境调试
