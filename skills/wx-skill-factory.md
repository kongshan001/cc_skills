# Skill Factory - 技能工厂

## 技能描述

**Skill Factory** 是技能工厂，AI作为规划层循环迭代生产新Skill（需求对齐→选档→设计→生成→测试→对比→迭代→交付），支持手动/半自动/全自动三档模式。

- **最新版本**: 1.0.0
- **作者**: mwangxiang
- **更新时间**: 2026-04-21
- **标签**: factory, automation, generation

## 功能列表

- 需求对齐
- 技能选档
- 自动设计生成
- 测试对比
- 迭代优化
- 交付发布
- 手动/半自动/全自动模式

## 安装方式

```bash
clawhub install wx-skill-factory
```

|------|------|------|

| ECS 服务器 | ✅ 适合 | 自动化生产 |
