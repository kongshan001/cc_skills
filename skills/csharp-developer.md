# C# Developer

> C# 开发技能

## 技能描述

使用 C# 构建 .NET 8+ 应用、ASP.NET Core API 或 Blazor Web 应用。提供 Entity Framework Core、最小 API、异步模式、CQRS 支持。

## 功能列表

- .NET 8+ 开发
- ASP.NET Core API
- Blazor 开发
- Entity Framework Core
- 最小 API
- 异步编程模式
- CQRS 架构

## 安装方式

```bash
clawhub install csharp-developer
```

## 推荐安装评估

- **本地环境**: ⭐⭐⭐⭐⭐ 适合 C# 开发
- **ECS**: ⭐⭐⭐⭐ 远程开发支持
