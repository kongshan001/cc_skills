# Skill Recommender - 技能推荐器

## 技能描述

**Skill Recommender** 按意图、功能或用例查找、过滤、聚类和推荐相似的 OpenClaw Skills。用于找同类技能、比较多个相近技能、判断哪个技能更好。

- **最新版本**: 0.1.0
- **作者**: harrylabsj
- **更新时间**: 2026-04-21
- **标签**: recommendation, comparison, clustering

## 功能列表

- 按意图搜索技能
- 功能相似度过滤
- 技能聚类分析
- 智能推荐
- 技能对比

## 安装方式

```bash
clawhub install skill-recommender
```

## 适用场景

- 找同类技能
- 技能对比选择
- 智能推荐
