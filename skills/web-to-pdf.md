# Web to PDF - 网页转 PDF

## 基本信息

- **技能名称**: web-to-pdf
- **版本**: 1.0.0

- **创建时间**: 2026-03-15

## 技能描述

Convert any webpage to a clean, high-quality PDF document with customizable options.

将任何网页转换为干净、高质量的 PDF 文档，支持自定义选项。

## 功能列表

- 网页转 PDF
- 高质量输出
- 自定义转换选项
- 支持复杂网页布局
- 保持样式和格式
