# Agency Agents — AI Specialists

## 技能描述

Agency Agents 是由 adisinghstudent 开发的 AI 专家代理集合，提供 50+ 个专业的 AI 代理人格，涵盖工程、设计、营销、销售等领域。每个代理都有深度领域专业知识、独特个性、定义的工作流程和可衡量的交付物。

## 安装方式

```bash
# 克隆仓库
git clone https://github.com/msitarzewski/agency-agents.git
cd agency-agents

# Claude Code 安装
cp -r agency-agents/* ~/.claude/agents/

# 或使用符号链接
ln -s /path/to/agency-agents ~/.claude/agents/agency
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 推荐 | 多领域专家辅助开发 |
| ECS/服务器 | ⚠️ 可选 | 代码相关功能可在服务器使用 |

### 本地部署优势
- 无需 API Key
- 工具无关 (Claude Code, Cursor, Aider, Windsurf, Copilot)
- 可扩展 - 添加自定义代理
- 社区活跃 (51,000+ stars)

### 特性
- 50+ 专业代理人格
- 深度领域专业知识
- 定义的工作流程
- 可衡量的交付物
- 开源免费
