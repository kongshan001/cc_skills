# Docker Manager

- **Slug**: docker-manager
- **作者**: sxliuyu
- **版本**: 1.0.2
- **更新时间**: 2026-03-17

## 背景需求

Docker 容器需要可视化管理和监控。

## 目标

提供 Docker 容器管理和监控功能。

## 功能列表

- 查看容器
- 启动/停止/删除容器
- 监控资源使用
- 查看日志
- 运维和开发场景

## 安装方式

```bash
clawhub install docker-manager
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐ 容器管理
- **ECS/服务器**: ⭐⭐⭐⭐⭐ 运维监控
