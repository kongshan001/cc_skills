# Meow Finder

## 技能描述

CLI 工具，用于发现 AI 工具。可按类别、价格和使用案例搜索 40+ 精选 AI 工具。

## 功能列表

- 按类别搜索 AI 工具
- 按价格筛选
- 按使用案例搜索
- 工具发现与推荐

## 安装方式

```bash
clawhub install meow-finder
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | 随时查找 AI 工具 |
| ECS | ⭐⭐⭐⭐ | 开发环境必备 |

## 适用场景

- 寻找特定 AI 工具替代品
- 发现新的 AI 解决方案
- 比较工具功能和定价
