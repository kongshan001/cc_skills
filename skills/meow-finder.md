## 描述

CLI tool to discover AI tools. Search 40+ curated tools by category, pricing, and use case.

CLI工具，用于发现AI工具。按类别、价格和使用场景搜索40+精选工具。

## 功能列表

- AI工具发现
- 按类别筛选
- 按价格筛选
- 按使用场景搜索
- 40+精选工具库
