# AI Dev Runtime

## 技能描述

AI Dev Runtime 工具集 - read_file、search、edit、run_terminal、run_tests，支持混合语义+关键词搜索和学习记忆。

## 功能列表

- 文件读取
- 语义+关键词搜索
- 代码编辑
- 终端运行
- 测试执行
- 学习记忆

## 安装方式

```bash
clawhub install ai-dev-runtime
```

- AI 编码辅助
- 代码搜索与导航
- 测试自动化
