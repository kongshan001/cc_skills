# MCP Review

## 技能描述

审查 MCP Server 工具实现是否符合接口设计准则。当用户要求 review、检查、审查 MCP 工具定义时说"check一下工具设计"、"review mcp tools"、"工具设计有没有问题"时使用。

## 功能列表

- MCP 接口审查
- 工具定义检查
- 设计规范验证
- 问题诊断

## 安装方式

```bash
clawhub install mcp-review
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | MCP 开发必备 |
| ECS | ⭐⭐⭐⭐ | CI 检查 |

## 适用场景

- MCP 工具开发
- 接口规范审查
- 代码审查流程
