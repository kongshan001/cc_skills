# Category Gating Guide — 亚马逊类目准入指南

## 背景需求

亚马逊卖家需要了解各类目的批准要求和gating流程，准备所需文档。

## 目标

导航亚马逊类目批准和gating要求，提供逐步解封流程、文档清单和常见拒绝解决方案。

## 设计方案

- 类目gating流程指南
- 所需文档清单
- 拒绝原因分析

## 本地部署

```bash
npm i -g clawhub
clawhub install category-gating-guide
```

## 推荐安装

**本地**：✅ 适合 - 静态指南类

## 效果展示

- 类目要求列表
- 申请流程指南

## 优缺点分析

**优点**：
- 覆盖常见类目
- 实用文档模板

**缺点**：
- 亚马逊政策常变
