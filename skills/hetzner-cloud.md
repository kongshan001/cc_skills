# Hetzner Cloud CLI

## 技能概述

- **Slug**: hetzner-cloud
- **名称**: Hetzner Cloud CLI
- **作者**: pasogott
- **版本**: 1.0.0
- **更新时间**: 2026-02-25

## 背景需求

Hetzner Cloud 是性价比高的云服务提供商，需要通过 CLI 管理服务器、卷、防火墙、网络等资源。

## 目标

Hetzner Cloud CLI for managing servers, volumes, firewalls, networks, DNS, and snapshots.

## 功能列表

- 服务器管理
- 卷管理
- 防火墙管理
- 网络管理
- DNS 管理
- 快照管理

## 安装方式

```bash
clawhub install hetzner-cloud
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐ 推荐
- **ECS 服务器**: ⭐⭐⭐⭐⭐ 强烈推荐

**理由**: 适合使用 Hetzner 作为基础设施的团队，可以高效管理云资源。

## 优缺点分析

### 优点

1. 高性价比
2. 全面的资源管理
3. 自动化运维
4. 成本优化

### 缺点

1. 限于 Hetzner 平台
2. 国内访问可能较慢
