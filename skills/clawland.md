# Clawland - Solana链上博弈游戏

## 技能描述

Play on-chain odd/even games on Solana devnet via Clawland. Mint GEM from SOL or USDC, bet odd or even, win 2x.

脚本处理钱包设置、铸造和自动游戏。

## 功能列表

- Solana devnet 链上游戏
- SOL/USDC 铸造 GEM 代币
- 奇偶投注 (2倍回报)
- 钱包自动化
- 自动游戏模式

## 安装方式

```bash
clawhub install clawland
```
