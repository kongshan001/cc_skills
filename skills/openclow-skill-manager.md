# OpenClaw Skill Manager - 技能管理器

## 技能描述

**OpenClaw Skill Manager** 管理已安装的 OpenClaw Skills：以表格形式列出所有 skills、显示功能介绍、找出功能重复的 skills。

- **最新版本**: 1.0.1
- **作者**: damiencronw
- **更新时间**: 2026-03-08
- **标签**: management, tools

## 功能列表

- 列出所有已安装技能
- 显示技能功能介绍
- 识别重复功能技能
- 表格化展示

## 安装方式

```bash
clawhub install openclow-skill-manager
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ✅ 强烈推荐 | 管理技能必备 |
| ECS 服务器 | ✅ 适合 | 服务器技能管理 |

## 适用场景

- 查看已安装技能
- 技能功能查询
- 清理重复技能
