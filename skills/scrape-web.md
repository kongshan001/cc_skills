# Scrape Web

## 技能概述

**Slug**: `scrape-web`  
**作者**: jnmhub  
**版本**: 1.0.0  
**更新日期**: 2026-03-17

## 背景需求

使用 Python + Scrapling 获取网页内容，支持简单选择器。

## 功能列表

1. Python Scrapling 爬虫
2. CSS 选择器支持
3. 简单网页内容提取

## 安装方式

```bash
clawhub install scrape-web
```
