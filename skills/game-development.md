## 技能描述

设计并交付可从零构建的 Three.js 原型到高级架构的浏览器游戏，包含工作流、预算和测试循环。

## 功能列表

- Three.js 原型快速开发
- 游戏架构设计
- 预算和项目管理
- 游戏测试循环
- 浏览器兼容优化

## 安装方式

```bash
clawhub install game-development
```
