# Devtopia

## 技能描述

使用 Devtopia CLI 管理 AI 构建工具的技能，用于在 agent 驱动的生态系统内发现、运行、创建和提交工具。

## 功能列表

- 工具发现
- 工具运行
- 工具创建
- 工具提交
- 生态系统集成

## 安装方式

```bash
clawhub install devtopia
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | 工具开发者 |
| ECS | ⭐⭐⭐⭐ | CLI 环境 |

## 适用场景

- 构建 AI 工具
- 工具市场发布
- 工具生态系统参与
