# Game AI Systems

## 技能描述

Game AI 开发指南，涵盖行为树、状态机、寻路和决策系统。用于实现敌人AI、NPC行为、战斗逻辑等游戏智能系统。

## 功能列表

- 行为树 (Behavior Trees) 实现
- 状态机 (State Machines) 模式
- 寻路算法 (Pathfinding)
- 决策系统 (Decision Making)
- 敌人AI实现
- NPC行为模式

## 安装方式

```bash
clawhub install game-ai
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 轻量级技能，本地直接使用 |
| ECS服务器 | ⭐⭐⭐⭐⭐ | 无外部依赖，纯逻辑处理 |

**推荐：本地开发环境**

- 无外部API依赖
- 纯逻辑实现，响应快速
- 适合游戏开发调试
