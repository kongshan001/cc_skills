# Game AI Systems

> Game AI development guide covering behavior trees, state machines, pathfinding, and decision-making systems. Use when implementing enemy AI, NPC behaviors, flocking, and game logic.

## 技能概述

Game AI Systems 提供游戏人工智能开发指南，涵盖行为树、状态机、寻路和决策系统。

## 功能列表

- 行为树 (Behavior Trees)
- 状态机 (State Machines)
- 寻路系统 (Pathfinding)
- 决策系统
- NPC 行为
- 群体行为 (Flocking)
- 游戏逻辑

## 安装方式

```bash
clawhub install game-ai
```
