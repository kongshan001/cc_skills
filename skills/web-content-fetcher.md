# Web Content Fetcher (网页内容获取工具)

## 技能概述

**Slug**: `web-content-fetcher`  
**作者**: mrtommywu  
**版本**: 1.0.1  
**更新日期**: 2026-04-11

## 背景需求

当使用常规爬虫被网站反爬机制过滤时，需要使用替代服务获取网页内容。

## 功能列表

1. **r.jina.ai** - 最稳定的网页内容获取服务
2. **markdown.new** - Cloudflare 专用解决方案
3. **defuddle.md** - 备用方案

## 触发词

- 获取网页内容
- 网页转markdown
- 内容抓取
- fetch webpage
- bypass 爬虫

## 安装方式

```bash
clawhub install web-content-fetcher
```

## 推荐安装评估

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 轻量级工具，本地安装即可使用 |
| ECS 服务器 | ⭐⭐⭐⭐⭐ | 无需额外依赖，直接调用外部服务 |

## 优缺点分析

**优点**:
- 集成多个备用方案，稳定性高
- 无需复杂配置
- 支持多种反爬场景

**缺点**:
- 依赖外部服务
- 可能存在请求频率限制
