# package-version-tracker (包版本追踪)

## 技能描述

查询 npm 和 PyPI 包的版本信息、历史发布及依赖，支持版本比较和批量查询，响应快速无须 API key。

## 功能列表

- npm 包版本查询
- PyPI 包版本查询
- 版本历史查询
- 依赖信息查询
- 版本比较
- 批量查询
- 无需 API key

## 安装方式

```bash
clawhub install package-version-tracker
```
