# Play Any Game - AI游戏伴侣助手

## 技能描述

AI游戏伴侣助手 - 当你在游戏中遇到困难、卡关、不知道怎么操作时，AI可以帮你分析游戏画面、解答问题、甚至简单操作界面帮你解决问题。

不是全自动代肝，而是你的游戏伙伴，在你需要的时候伸出援手。

触发关键词：帮我看下这个怎么过、这个怎么操作、卡关了、不知道怎么弄

支持原神、崩坏星穹铁道等各类游戏。

## 功能列表

- 游戏画面分析
- 关卡攻略指导
- 操作问题解答
- 游戏伙伴陪伴模式

## 安装方式

```bash
clawhub install play-any-game
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐ | 需要配合游戏画面输入 |
| ECS服务器 | ⭐⭐⭐⭐⭐ | 适合游戏玩家群聊使用 |

**推荐：ECS服务器**

- 适合游戏玩家社群使用
- 支持多种主流游戏
- 伴侣式辅助模式有差异化
