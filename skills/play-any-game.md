# AI游戏伴侣 (play-any-game)

## 背景需求

玩家在游戏中遇到困难、卡关时需要帮助，但不想找人代练也不想看冗长的攻略。AI 可以实时分析游戏画面、解答问题、提供指导。

## 目标

作为玩家的游戏伙伴，在需要时提供游戏指导、操作建议、问题解答，支持原神、崩坏星穹铁道等各类游戏。

## 设计方案

- **核心功能**:
  - 分析游戏画面，提供指导
  - 解答游戏问题
  - 操作界面辅助（非全自动代肝）
- **触发方式**: "帮我看下这个怎么过"、"这个怎么操作"、"卡关了"、"不知道怎么弄"
- **支持游戏**: 原神、崩坏星穹铁道等各类游戏

## 本地部署

```bash
clawhub install play-any-game
```

## 效果展示

- 实时游戏指导
- 画面分析与建议
- 玩家伙伴式陪伴

## 优缺点分析

✅ 优点:
- 场景覆盖广，多游戏支持
- 非代肝，保留玩家成就感
- 实时响应

❌ 缺点:
- 需要游戏画面输入（截图/描述）
- 复杂操作仍需玩家执行

## 平替对比

| 特性 | play-any-game | 攻略网站 |
|------|---------------|----------|
| 响应方式 | 实时问答 | 搜索浏览 |
| 个性化 | 高 | 低 |
| 交互形式 | 对话 | 文档 |

## 落地过程

1. 游戏中遇到问题时描述或截图
2. AI 分析并给出指导
3. 玩家根据建议继续游戏
