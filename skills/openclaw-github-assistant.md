# OpenClaw GitHub Assistant

## 技能描述

查询和管理 GitHub 仓库 - 列出仓库、检查 CI 状态、创建 Issues、搜索仓库、查看最近活动。

## 功能列表

- 仓库列表查询
- CI 状态检查
- Issues 创建和管理
- 仓库搜索
- 最近活动查看
- 仓库信息获取

## 安装方式

```bash
clawhub install openclaw-github-assistant
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | GitHub 集成必备 |
| ECS/服务器 | ⭐⭐⭐⭐ | 自动化 GitHub 操作 |
