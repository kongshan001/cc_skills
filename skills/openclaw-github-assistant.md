# OpenClaw GitHub Assistant

## 技能概述

- **Slug**: openclaw-github-assistant
- **作者**: conorkenn
- **版本**: 2.0.1
- **更新时间**: 2026-04-24
- **标签**: github, assistant, cli

## 功能列表

1. 查询和管理 GitHub 仓库
2. 列出仓库
3. 检查 CI 状态
4. 创建 Issues
5. 搜索仓库
6. 查看最近活动

## 安装方式

```bash
clawhub install openclaw-github-assistant
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 轻量级工具，适合本地 GitHub 日常操作 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 适合 CI/CD 自动化流程 |

## 适用场景

- 开发者日常 GitHub 仓库管理
- CI/CD 状态监控
- Issue 自动化创建和管理
- 仓库搜索和探索

## 优缺点分析

**优点**:
- 功能全面，覆盖 GitHub 常用操作
- 轻量级，易于集成
- 持续更新维护

**缺点**:
- 需要 GitHub Token 认证
- 部分高级功能依赖 API 限制
