## 技能描述

Query and manage GitHub repositories - list repos, check CI status, create issues, search repos, and view recent activity.

查询和管理 GitHub 仓库 - 列出仓库、检查 CI 状态、创建 issues、搜索仓库和查看最近活动。

## 功能列表

- 列出 GitHub 仓库
- 检查 CI/CD 状态
- 创建和管理 Issues
- 搜索仓库
- 查看最近活动记录
