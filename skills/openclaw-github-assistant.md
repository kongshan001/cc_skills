# OpenClaw GitHub Assistant

- **Slug**: openclaw-github-assistant
- **作者**: conorkenn
- **版本**: 2.0.1
- **更新时间**: 2026-04-14

## 背景需求

在 AI Agent 环境中管理 GitHub 仓库，查看 CI 状态、创建 Issue 等。

## 目标

提供 GitHub 仓库查询和管理的自动化能力。

## 功能列表

- 列出仓库
- 查看 CI 状态
- 创建 Issue
- 搜索仓库
- 查看最近活动

## 安装方式

```bash
clawhub install openclaw-github-assistant
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 工作流自动化
- **ECS/服务器**: ⭐⭐⭐⭐ 需配置 GitHub Token
