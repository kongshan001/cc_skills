# Skill Finder CN Pro - 技能查找器专业版

## 技能描述

**Skill Finder CN Pro** 是中文技能查找器，帮助发现和安装 ClawHub Skills。回答"有什么技能可以X"、"找一个技能"等问题。

- **最新版本**: 1.0.0
- **作者**: lvjunjie-byte
- **更新时间**: 2026-04-21
- **标签**: chinese, search, discovery

## 功能列表

- 中文技能搜索
- ClawHub 技能发现
- 智能技能推荐
- 安装辅助

## 安装方式

```bash
clawhub install skill-finder-cn-pro
```
