# Agency Frontend Developer 技能调研报告

## 1. 背景需求

代理机构（Agency）需要快速交付高质量前端项目。相比企业内部开发，代理机构面临更紧迫的时间线和多样化的客户需求。

## 2. 目标

为代理机构提供前端开发能力：
- 快速原型开发
- 客户需求适配
- 可复用组件库
- 品牌定制

## 3. 设计方案

### 核心能力
- **快速原型**: 短时间内交付可用原型
- **组件复用**: 跨项目共享 UI 组件
- **品牌适配**: 快速切换主题和品牌
- **响应式设计**: 多设备兼容

### 技术栈
- Next.js / Nuxt.js
- Tailwind CSS
- Figma to Code
- Storybook

## 4. 本地部署

clawhub install agency-frontend-developer

## 5. 效果展示

典型应用场景：
- 客户网站开发
- 营销落地页
- 品牌官网
- 活动页面

## 6. 优缺点分析

### 优点
- 快速交付能力
- 高度可定制
- 组件复用

### 缺点
- 可能过度依赖模板
- 复杂功能需额外开发

## 7. 平替对比

| 技能 | 特点 |
|------|------|
| agency-frontend-developer | 代理机构快速交付 |
| expert-frontend-developer | 专家级深度开发 |
