# GitHub CLI

## 技能概述

- **Slug**: github-cli
- **名称**: Github Cli
- **作者**: tag-assistant
- **版本**: 1.0.0
- **更新时间**: 2026-04-09

## 背景需求

GitHub CLI (gh) 是强大的命令行工具，但命令众多难以记忆。需要一个全面的参考指南。

## 目标

Comprehensive GitHub CLI (gh) reference. Covers repos, issues, PRs, Actions, releases, gists, search, projects v2, API, secrets/variables, labels, codespaces.

## 功能列表

- 仓库操作参考
- Issue 管理
- PR 管理
- Actions 工作流
- Release 管理
- Gist 操作
- 搜索功能
- Projects v2
- API 调用
- Secrets/Variables 管理
- Labels 管理
- Codespaces 管理

## 安装方式

```bash
clawhub install github-cli
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS 服务器**: ⭐⭐⭐⭐⭐ 强烈推荐

**理由**: GitHub CLI 是开发者必备工具，此技能提供全面的命令参考。

## 优缺点分析

### 优点

1. 全面的 gh 命令参考
2. 提高命令行效率
3. 覆盖几乎所有 GitHub 功能
4. 减少查阅文档时间

### 缺点

1. 需要安装 gh CLI
2. 需要 GitHub 认证
