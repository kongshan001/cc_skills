# GitHub CLI

- **Slug**: github-cli
- **作者**: tag-assistant
- **版本**: 1.0.0
- **更新时间**: 2026-04-14

## 背景需求

GitHub CLI (gh) 是管理 GitHub 资源的标准工具。

## 目标

提供全面的 gh 命令参考，覆盖日常 GitHub 操作。

## 功能列表

- 仓库操作 (repos)
- Issue 管理
- PR 操作
- Actions 工作流
- Releases
- Gists
- 搜索
- Projects v2
- API 调用
- Secrets/变量
- Labels
- Codespaces

## 安装方式

```bash
clawhub install github-cli
```
