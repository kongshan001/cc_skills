# GitHub CLI

## 技能描述

全面的 GitHub CLI (gh) 参考技能，涵盖仓库管理、Issues、PRs、Actions、Releases、Gists、搜索、项目管理 v2、API、Secrets/Variables、Labels、Codespaces 等。

## 功能列表

- 仓库创建和管理
- Issues 创建、列表、编辑
- PR 创建、审查、合并
- GitHub Actions 工作流管理
- Releases 和 Gists 操作
- 项目管理 v2 (Projects)
- GitHub API 调用
- Secrets 和 Variables 管理
- Labels 管理
- Codespaces 操作

## 安装方式

```bash
clawhub install github-cli
```
