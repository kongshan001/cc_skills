# COCOS CLI Game Development

| 项目 | 内容 |
|------|------|
| **技能名称** | cocos-cli-game-dev |
| **所有者** | liuyihunter |
| **版本** | 0.0.3 |
| **更新时间** | 2026-04-07 |

## 技能描述

创建、构建和修改 Cocos 游戏项目。用于用户请求创建新 Cocos 游戏（2D/3D）或修改现有游戏。

## 功能列表

- Cocos 游戏项目创建
- 2D/3D 游戏开发
- 项目构建和打包
- 游戏修改和扩展

## 安装方式

```bash
clawhub install cocos-cli-game-dev
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| **本地开发** | ⭐⭐⭐⭐⭐ | Cocos CLI 本地运行 |
| **ECS服务器** | ⭐⭐ | 需要图形环境构建 |

## 使用场景

- Cocos 2D/3D 游戏开发
- 移动游戏项目
