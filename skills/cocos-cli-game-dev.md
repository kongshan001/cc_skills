# COCOS CLI Game Development

> Cocos 游戏开发工具

## 技能描述

创建、构建和修改 Cocos 游戏项目。当用户想要启动新游戏（2D/3D）或修改现有游戏时触发。支持 Cocos Creator。

## 核心功能

1. **新建游戏项目** - Create Cocos game
2. **游戏构建** - Build game
3. **游戏修改** - Modify existing game
4. **2D/3D 支持** - 支持 2D 和 3D 游戏

## 安装方式

```bash
clawhub install cocos-cli-game-dev
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 游戏开发者必备 |
| ECS/服务器 | ⭐⭐⭐ | 很少在服务器开发 |
| 个人项目 | ⭐⭐⭐⭐⭐ | 独立游戏开发 |

### 适用人群

- Cocos 游戏开发者
- 独立游戏开发者
- 2D/3D 游戏开发团队
