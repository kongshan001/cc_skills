# Docker Diag

> Advanced log analysis for Docker containers using signal extraction.

## 技能概述

Docker Diag 提供容器日志高级分析能力，使用信号提取技术进行问题诊断。

## 功能列表

- 容器日志分析
- 信号提取
- 问题诊断
- 日志可视化

## 安装方式

```bash
clawhub install docker-diag
```
