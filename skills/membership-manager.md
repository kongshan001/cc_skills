# Membership Manager - 会员管理

## 基本信息

- **技能名称**: membership-manager
- **版本**: 1.0.0
- **作者**: harrylabsj
- **创建时间**: 2026-03-15
- **更新时间**: 2026-03-15

## 技能描述

Manage ecommerce and local-service memberships, benefit tracking, renewal reminders, and cancellation guidance.

管理电商和本地服务会员、权益追踪、续费提醒和取消指导。

## 功能列表

- 电商会员管理
- 本地服务会员管理
- 权益追踪
- 续费提醒
- 取消指导

clawhub install membership-manager
