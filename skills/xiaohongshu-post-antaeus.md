# xiaohongshu-post-antaeus (小红书自动发帖)

## 技能描述

Create and publish Xiaohongshu (小红书/RED) notes. Use when: user wants to publish a note to Xiaohongshu, schedule or batch post content, generate Xiaohongshu-s...

创建和发布小红书笔记。支持定时发布、批量发布内容，生成小红书风格的帖子。

## 功能列表

- 创建小红书笔记
- 发布笔记到小红书
- 定时发布支持
- 批量发布内容
- 小红书编辑器优化（段落标签处理）

## 安装方式

```bash
clawhub install xiaohongshu-post-antaeus
```
