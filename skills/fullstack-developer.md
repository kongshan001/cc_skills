# Fullstack Developer

> 全栈开发技能

## 技能描述

World-class fullstack development skill，覆盖前端(React, Next.js, Vue, HTML/CSS/JS)、后端(Node.js, Python/FastAPI, Django, Express)、数据库等全栈技术。

## 功能列表

- React/Next.js 前端开发
- Vue 前端开发
- Node.js 后端开发
- Python/FastAPI 后端开发
- Django 框架开发
- Express 开发
- 数据库设计与集成
- DevOps 部署

## 安装方式

```bash
clawhub install fullstack-developer
```

## 推荐安装评估

- **本地环境**: ⭐⭐⭐⭐⭐ 适合全栈项目开发
- **ECS**: ⭐⭐⭐⭐ 远程开发支持
