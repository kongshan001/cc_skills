## 技能描述

通过 inference.sh 在安全沙箱环境中执行 Python 代码。预装：NumPy、Pandas、Matplotlib、requests、BeautifulSoup。

## 功能列表

- Python 代码安全沙箱执行
- 预装科学计算库（NumPy、Pandas）
- 数据可视化（Matplotlib）
- HTTP 请求（requests）
- HTML 解析（BeautifulSoup）

## 安装方式

```bash
clawhub install python-executor
```
