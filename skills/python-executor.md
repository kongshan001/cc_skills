# Python Executor

- **Slug**: python-executor
- **作者**: okaris
- **版本**: 0.1.5
- **更新时间**: 2026-04-14

## 背景需求

在 AI Agent 环境中执行 Python 代码进行数据分析、脚本生成、计算等任务。

## 目标

提供安全的沙箱环境执行 Python 代码，支持常用数据科学库。

## 功能列表

- Python 代码沙箱执行
- 预装库: NumPy, Pandas, Matplotlib, requests, BeautifulSoup 等
- 安全的推理执行环境

## 安装方式

```bash
clawhub install python-executor
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 轻量级，快速集成
- **ECS/服务器**: ⭐⭐⭐⭐⭐ 支持远程执行

- 直接使用本地 Python 环境
