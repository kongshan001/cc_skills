# Python Executor

## 技能描述

在安全的沙箱环境中执行 Python 代码，预装 NumPy、Pandas、Matplotlib、requests、BeautifulSoup 等常用库。

## 功能列表

- 安全沙箱 Python 执行
- 预装科学计算库（NumPy, Pandas）
- 预装数据可视化（Matplotlib）
- 预装网络请求库（requests, BeautifulSoup）
- 代码执行结果返回

## 安装方式

```bash
clawhub install python-executor
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐ | 需要执行简单 Python 脚本时使用 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 服务器端数据处理和脚本执行 |

## 使用场景

- 数据分析任务
- 自动化脚本执行
- API 数据抓取
- 快速原型验证
