## 技能描述

Execute Python code in a safe sandboxed environment via inference.sh. Pre-installed: NumPy, Pandas, Matplotlib, requests, BeautifulSoup, Pillow, Scipy, Scikit-learn, PyTorch, TensorFlow.

通过 inference.sh 在安全的沙箱环境中执行 Python 代码。预装：NumPy、Pandas、Matplotlib、requests、BeautifulSoup、Pillow、Scipy、Scikit-learn、PyTorch、TensorFlow。
