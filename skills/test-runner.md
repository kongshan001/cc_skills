# Test Runner

## 技能描述

使用推荐框架编写、运行和管理 TypeScript、Python 和 Swift 的单元测试、集成测试和 E2E 测试。

## 功能列表

- 单元测试编写和运行
- 集成测试编写和运行
- E2E 测试编写和运行
- TypeScript 测试支持
- Python 测试支持
- Swift 测试支持
- 测试框架推荐

## 安装方式

```bash
clawhub install test-runner
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 开发测试必备 |
| ECS/服务器 | ⭐⭐⭐⭐ | CI/CD 测试运行 |
