# Docker Sandbox

- **Slug**: docker-sandbox
- **作者**: gitgoodordietrying
- **版本**: 1.0.0
- **更新时间**: 2026-02-28

## 背景需求

运行不可信代码、探索包或隔离 Agent 工作负载需要安全沙箱环境。

## 目标

为 Agent 提供安全的 Docker 沙箱 VM 环境。

## 功能列表

- Docker 沙箱环境创建和管理
- 支持 Claude、Codex、Copilot、 Gemini、Kiro 等 Agent
- 网络代理控制
- 安全隔离执行

## 安装方式

```bash
clawhub install docker-sandbox
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 安全测试
- **ECS/服务器**: ⭐⭐⭐⭐⭐ 多租户隔离
