# Docker Sandbox

## 技能概述

- **Slug**: docker-sandbox
- **作者**: gitgoodordietrying
- **版本**: 1.0.0
- **更新时间**: 2026-02-28
- **标签**: docker, sandbox, security, isolation

## 功能列表

1. 创建和管理 Docker 沙箱环境
2. 安全执行不可信代码
3. 探索和测试软件包
4. 隔离 Agent 工作负载
5. 支持 Claude、Codex、Copilot、 Gemini、Kiro agents
6. 网络代理控制

## 安装方式

```bash
clawhub install docker-sandbox
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 安全测试必备 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 隔离执行环境 |

## 适用场景

- 运行不可信代码
- 软件包安全探索
- Agent 隔离执行
- 恶意软件分析（隔离环境）

## 优缺点分析

**优点**:
- 强大的隔离机制
- 多 Agent 支持
- 网络控制灵活

**缺点**:
- 需要 Docker 运行环境
- 资源占用相对较高
