# Cli Developer

## 技能描述

CLI Developer 是一个用于构建 CLI 工具的专业技能。当你需要构建 CLI 工具、实现参数解析或添加交互式提示时使用。

## 功能列表

- CLI 设计指导
- 参数解析实现
- 交互式提示
- 进度指示器
- Shell 自动补全

## 安装方式

```bash
clawhub install cli-developer
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | 开发者本地 CLI 开发首选 |
| ECS | ⭐⭐⭐ | 需要图形终端支持 |

## 适用场景

- 构建新的命令行工具
- 实现用户友好的 CLI 交互
- 添加自动补全功能
