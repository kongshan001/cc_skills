# Game Developer Skill 技能调研报告

## 1. 背景需求

游戏开发是复杂的技术领域，涉及多个引擎和大量架构模式。Unity 和 Unreal 是最流行的游戏引擎，开发者需要专业的技术指导来实现游戏系统和优化性能。

## 2. 目标

为 Claude Code 提供游戏开发能力，涵盖：
- Unity 开发
- Unreal 开发
- 游戏设计模式
- ECS 架构
- 物理系统
- 性能优化

## 3. 设计方案

### 支持引擎
- **Unity**: C# 脚本、组件系统、资产管理
- **Unreal**: C++/Blueprint、Actor 系统

### 核心能力
- 游戏系统架构设计
- ECS（实体组件系统）实现
- 物理系统集成
- 性能分析与优化

### 技术栈
- Unity 2023+
- Unreal Engine 5
- C# / C++

## 4. 本地部署

```bash
# 安装 Game Developer Skill 技能
clawhub install game-developer-skill
```

## 5. 效果展示

典型应用场景：
- 创建游戏核心机制
- 实现玩家控制系统
- 集成物理交互
- 优化游戏性能

## 6. 优缺点分析

### 优点
- 覆盖主流游戏引擎
- 包含高级架构模式（ECS）
- 性能优化指导

### 缺点
- 需要游戏开发基础
- 移动端游戏支持有限

## 7. 平替对比

| 技能 | 特点 |
|------|------|
| game-developer-skill | Unity/Unreal 游戏开发 |
| android-3d-developer | Android 3D 开发 |

## 8. 落地过程

1. 执行 `clawhub install game-developer-skill`
2. 描述游戏项目需求
3. 根据指导选择引擎和架构
4. 实现具体游戏系统
