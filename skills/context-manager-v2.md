# context-manager-v2 (上下文管理器)

## 技能描述

Auto context management with seamless session switching. Monitors usage, triggers at 85% threshold, automatically creates new session with loaded memory. Zero config...

自动上下文管理，无缝会话切换。监控使用率，在 85% 阈值时触发，自动创建新会话并加载记忆。零配置启动。

## 功能列表

- 上下文使用监控
- 85% 阈值自动触发
- 自动创建新会话
- 记忆自动加载
- 增强的错误处理
- 无缝会话切换
