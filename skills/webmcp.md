# WebMCP

## 技能描述

当浏览或自动化通过 WebMCP API (window.navigator.modelContext) 暴露工具的网页时使用。教 agents 如何发现、检查和调用网页上的 WebMCP 工具，而非依赖 DOM 抓取或 UI 执行。

## 功能列表

- WebMCP 工具发现
- 工具检查与调用
- API 上下文检测
- 无需 DOM 抓取

## 安装方式

```bash
clawhub install webmcp
# 或
clawhub install web-mcp
```

- 自动化 Web 交互
- 构建 AI Browser 代理
