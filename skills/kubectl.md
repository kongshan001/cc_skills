## 技能描述

通过 kubectl 命令执行和管理 Kubernetes 集群。查询资源、部署应用、调试容器、管理配置、监控集群健康。

## 功能列表

- 资源查询
- 应用部署
- 容器调试
- 配置管理
- 集群监控

## 安装方式

```bash
clawhub install kubectl
```
