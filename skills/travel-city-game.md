# Travel City Game

> 旅行游戏副本生成器

## 基本信息

- **Slug**: travel-city-game
- **作者**: hongweisong1999
- **版本**: 1.0.9
- **更新时间**: 2026-04-08

## 技能描述

根据用户输入的城市名，自动调用 flyai 搜索该城市的景点、体验、美食产品，然后生成一个游戏化的3节点叙事行程 H5 页面。每个节点包含 AI 生成的故事文案和飞猪真实预订链接。

## 功能列表

- 城市景点搜索
- 体验产品获取
- 美食推荐
- AI 故事文案生成
- 飞猪预订链接
- H5 页面生成
- 3节点叙事结构

## 使用方式

```
/travel-city-game 城市名
```

当用户提到 "travel-city-game"、"生成行程"、"旅行叙..." 时触发

## 安装方式

```bash
clawhub install travel-city-game
```
