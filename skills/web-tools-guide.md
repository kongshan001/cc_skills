# Web Tools Guide

## 技能概述

**Slug**: `web-tools-guide`  
**作者**: fmls  
**版本**: 1.0.1  
**更新日期**: 2026-03-31

## 背景需求

Web 工具策略指南。当用户提到搜索、上网、查资料、打开网站、抓取网页、获取网络信息、新闻、热点等关键词时触发。

## 功能列表

1. web_search 工具策略
2. web_fetch 工具策略
3. browser 工具策略
4. JS 渲染页面处理策略

## 触发词

- 搜索/上网/查资料/打开网站
- 抓取网页/获取网络信息
- web search/web fetch/browser

## 安装方式

```bash
clawhub install web-tools-guide
```

## 优缺点分析

**优点**:
- 提供工具使用策略指导
- 覆盖常见 Web 操作场景

**缺点**:
