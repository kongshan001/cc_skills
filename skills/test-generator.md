## 技能描述

Automated test case generator. Unit tests, integration tests, end-to-end tests, mock objects, test fixtures, coverage analysis, edge case generation.

自动化测试用例生成器。单元测试、集成测试、端到端测试、模拟对象、测试夹具、覆盖率分析、边界情况生成。

## 功能列表

- 单元测试自动生成
- 集成测试生成
- E2E 测试生成
- Mock 对象创建
- 测试夹具生成
- 覆盖率分析
- 边界情况生成
