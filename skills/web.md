## 背景需求

Web 开发是现代软件开发的核心。开发者需要使用 HTML、CSS、JavaScript 和现代框架来构建生产级别的网站和应用。

## 目标

提供全面的 Web 开发技能，帮助开发者构建、调试和部署网站，遵循生产最佳实践。

## 功能列表

1. **HTML/CSS/JS** - 基础 Web 技术
2. **现代框架** - React、Vue、Angular 等
3. **调试工具** - 浏览器开发者工具
4. **部署流程** - 网站部署最佳实践
5. **生产最佳实践** - 性能优化、安全性

## 安装方式

```bash
# 安装 Node.js (如未安装)
apt-get install nodejs npm

# 安装技能
clawhub install web
```
