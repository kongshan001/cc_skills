# Gamebox 多人游戏引擎 (gamebox)

## 背景需求

开发者希望快速构建多人游戏，但每款游戏单独开发工作量大。需要一个可复用的游戏引擎框架。

## 目标

提供多人游戏引擎框架，5 款游戏共用一套回合/状态/消息系统，降低开发成本。

## 设计方案

- **支持游戏**: 文字冒险、狼人杀、小说接龙、夺旗战、文明模拟
- **核心架构**:
  - 统一回合/状态/消息系统
  - LLM 负责叙事和动态内容
  - 脚本负责状态管理和规则执行
- **技术栈**: 纯 Python 标准库，零外部依赖
- **平台**: Windows/macOS/Linux
- **通信**: 共享目录，多 Agent 可同时参与

## 本地部署

```bash
clawhub install gamebox
```

## 效果展示

- 框架式游戏引擎
- 多游戏支持
- 零依赖部署

## 优缺点分析

✅ 优点:
- 框架复用，成本低
- 纯标准库，部署简单
- 多 Agent 支持

❌ 缺点:
- 需要 Python 环境
- 共享目录方式适合本地

## 落地过程

1. 安装技能
2. 选择想玩的游戏
3. 多人加入开始游戏
