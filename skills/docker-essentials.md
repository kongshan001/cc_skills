# Docker Essentials

## 技能描述

Docker Essentials 提供 Docker 容器管理的核心命令和工作流程，涵盖容器操作、镜像管理和故障排查。

## 功能列表

- 容器生命周期管理（创建、启动、停止、删除）
- 镜像操作（拉取、构建、推送、列出）
- 容器日志和调试
- 网络和存储配置
- Docker Compose 基础操作

## 安装方式

```bash
clawhub install docker-essentials
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 必备技能，本地 Docker 开发环境 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 服务器容器管理必备 |
