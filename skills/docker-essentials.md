# Docker Essentials

- **Slug**: docker-essentials
- **作者**: arnarsson
- **版本**: 1.0.0
- **更新时间**: 2026-02-26

## 背景需求

容器化是现代开发标配，需要快速掌握 Docker 常用命令和工作流。

## 目标

提供 Docker 容器管理、镜像操作、调试的常用命令参考。

## 功能列表

- 容器管理命令
- 镜像操作
- 调试和日志查看
- 卷和网络管理

## 安装方式

```bash
clawhub install docker-essentials
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 开发必备
- **ECS/服务器**: ⭐⭐⭐⭐⭐ 运维必备
