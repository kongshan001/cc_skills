# Lowcode Platform Development

> 低代码平台开发工具

## 技能描述

自动化创建开发所需角色 scaffolds 项目结构，生成 Vue2 + Element UI 的低代码开发平台代码。

## 核心功能

1. **角色自动化创建** - 自动创建开发角色
2. **项目结构脚手架** - 生成完整项目结构
3. **Vue2 + Element UI** - 基于 Vue2 和 Element UI
4. **低代码组件生成** - 生成低代码平台组件

## 安装方式

```bash
clawhub install lowcode-platform-development
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐ | 适合快速搭建后台系统 |
| ECS/服务器 | ⭐⭐⭐⭐ | 企业内部工具平台 |
| 个人项目 | ⭐⭐⭐ | 需要一定前端基础 |

### 适用人群

- 需要快速开发后台系统的开发者
- 低代码平台开发者
- Vue2 开发者
