# Lowcode Platform Development

## 技能描述

自动化创建所需开发角色、项目结构脚手架，并生成Vue2 + Element UI的低代码开发平台代码。适用于快速搭建低代码平台。

## 核心功能

- 自动化开发角色创建
- 项目结构脚手架生成
- Vue2 + Element UI低代码平台代码生成

## 安装方式

```bash
clawhub install lowcode-platform-development
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐ 较少使用

**理由**: 专注于前端脚手架生成，适合本地开发使用。
