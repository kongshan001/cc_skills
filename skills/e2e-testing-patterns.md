# E2E Testing Patterns

| 项目 | 内容 |
|------|------|
| **技能名称** | e2e-testing-patterns |
| **所有者** | wpank |
| **版本** | 1.0.0 |
| **更新时间** | 2026-02-26 |

## 技能描述

使用 Playwright 和 Cypress 构建可靠、快速的 E2E 测试套件，包含关键用户路径覆盖、 flaky 测试消除、CI/CD 集成。

## 功能列表

- Playwright 集成
- Cypress 集成
- 关键路径测试
- Flaky 测试消除
- CI/CD 集成

## 安装方式

```bash
clawhub install e2e-testing-patterns
```
