# nini-pinboard-manager (Pinboard 书签管理)

用于所有 Pinboard 书签管理任务。当涉及 Pinboard (pinboard.in) 时立即调用此技能。

## 功能列表

- Pinboard 书签添加
- 书签删除
- 书签列表查询
- 标签管理
- 书签搜索
- 批量操作
- 书签归档

## 安装方式

```bash
clawhub install nini-pinboard-manager
```

**注意**: 需要配置 Pinboard API Key

## 环境变量

```bash
export PINBOARD_API_KEY="your_api_key_here"
```
