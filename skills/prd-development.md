# PRD Development

## 技能描述

将产品需求转化为Coding Agent可执行的需求文档。当用户提到"写PRD"、"产品需求文档"、"用户故事"、"功能说明"，或提供会议记录/功能描述需要整理成开发文档时触发。

## 核心功能

- 产品需求文档编写
- 用户故事整理
- 会议记录转换为开发文档
- 功能描述规范化为开发文档

## 安装方式

```bash
clawhub install prd-development
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐⭐ 适用

**理由**: 专注于需求文档生成，适合产品经理和开发团队在本地环境使用。
