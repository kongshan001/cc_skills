# PRD Development

> 产品需求文档生成工具

## 技能描述

将产品需求转化为 Coding Agent 可执行的需求文档。当用户提到"写PRD"、"产品需求文档"、"用户故事"、"功能说明"，或提供会议记录/功能描述需要整理成开发文档时触发。

## 核心功能

1. **需求整理** - 将模糊需求转化为结构化文档
2. **用户故事撰写** - 编写清晰的用户故事
3. **功能说明** - 详细的功能描述
4. **会议记录转换** - 将会议讨论转化为开发文档

## 安装方式

```bash
clawhub install prd-development
```
