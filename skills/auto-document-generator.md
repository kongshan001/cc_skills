# auto-document-generator (自动文档生成)

## 技能描述

自动从代码生成技术文档，支持 Python/JavaScript/Bash，AI 增强文档质量

## 功能列表

- Python 代码文档生成
- JavaScript 代码文档生成
- Bash 脚本文档生成
- AI 增强文档质量
- 多语言支持
- 多种输出格式 (Markdown/HTML/PDF)
- 文件监控实时更新
- Git hooks 集成
- CI/CD 集成
- 自定义模板

## 安装方式

```bash
clawhub install auto-document-generator
```
