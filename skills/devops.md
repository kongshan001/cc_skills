# DevOps

## 技能描述

DevOps 是由 ivangdavila 开发的运维技能，涵盖 CI/CD 流水线、部署策略、基础设施即代码、容器化、 secrets 管理、监控告警和可靠性工程。

## 功能列表

### CI/CD 流水线
- 失败快速 - 单元测试先于集成测试
- 依赖缓存
- Pin action 版本 (SHA)
- 并行作业
- Secrets 管理

### 部署策略
- Blue-green - 并行运行、原子切换
- Canary - 百分比流量
- Rolling - 增量更新
- 回滚计划

### 基础设施即代码
- 版本控制基础设施
- Terraform plan/apply
- 状态文件安全
- 可重用模块
- 环境分离

### 容器化
- 每个容器一个进程
- 健康检查必需
- 非 root 用户运行
- 不可变镜像
- 使用 SHA 标记

### Secrets 管理
- 使用 vault/sealed secrets
- 定期轮换
- 环境隔离
- 审计访问

### 监控与告警
- 四个黄金信号: 延迟、流量、错误、饱和
- 告警症状而非原因
- 每个告警必须可操作
- 仪表板

### 可靠性
- SLO 定义
- 错误预算
- 混沌工程
- Runbook
- 无责复盘

### 常见错误
- SSH 登录修复
- 无预发布环境
- 忽略 flaky 测试
- 手动部署步骤
- 仅监控正常路径

### 网络
- 内部服务无需公网 IP
- TLS  everywhere
- DNS 服务发现
- 负载均衡健康检查
- 防火墙默认拒绝

## 安装方式

```bash
# 使用 ClawHub 安装
clawhub install devops
```

## 推荐安装评估

| 场景 | 推荐 | 说明 |
|------|------|------|
| 本地开发 | ⚠️ 可选 | 开发人员了解运维概念 |

- Docker Compose 本地开发
- Minikube/Kind 开发集群
- 本地 CI/CD 测试

### ECS 部署
- 云基础设施管理
- 生产部署流程
- 监控系统配置
