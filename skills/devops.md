## 背景需求

现代软件交付需要可靠的 CI/CD 流程和基础设施自动化。开发者需要工具来管理部署、基础设施和构建可靠的流水线。

## 目标

提供全面的 DevOps 自动化技能，帮助开发者实现部署自动化、基础设施管理和 CI/CD 流水线构建。

## 功能列表

1. **部署自动化** - 自动化部署流程
2. **基础设施管理** - 基础设施即代码
3. **CI/CD 流水线** - 构建可靠的持续集成/部署流水线
4. **多平台支持** - 支持各种云平台

## 安装方式

```bash
clawhub install devops
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐ 适合学习
- **ECS 服务器**: ⭐⭐⭐⭐⭐ 生产环境必备

## 优缺点分析

### 优点
- 功能全面
- 覆盖 DevOps 核心领域
- 适合自动化

### 缺点
- 需要一定的 DevOps 基础
- 部分功能需要云平台账号
