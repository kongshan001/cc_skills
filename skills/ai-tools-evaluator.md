# AI Tools Evaluator

## 技能描述

AI 工具评估器 - 评估和比较特定用例的 AI 工具。用于用户询问 AI 工具比较、AI 产品评测、工具推荐、ChatGPT 替代，或想找到最佳 AI 工具。

## 功能列表

- AI 工具评估
- 工具比较
- 推荐系统
- 用例分析

## 安装方式

```bash
clawhub install ai-tools-evaluator
```

| 环境 | 推荐度 | 说明 |
|------|--------|------|

## 适用场景

- AI 工具选型
- 产品评测
- 替代方案查找
