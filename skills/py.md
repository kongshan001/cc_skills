# Python Skill - 可靠Python编程

## 技能描述
专注于编写可靠Python的技能，帮助避免可变默认参数、导入陷阱和常见的运行时意外。提供Python编程的最佳实践和常见陷阱解决方案，确保代码的稳定性和可维护性。

## 功能列表
- **动态类型与类型提示**: 动态类型、类型提示、鸭子类型
- **集合操作**: 列表/字典/集合陷阱、推导式
- **函数编程**: 参数/关键字参数、闭包、装饰器、生成器
- **面向对象**: 继承、描述符、元类
- **并发编程**: GIL、线程、异步、多进程
- **模块导入**: 循环导入、包、__init__.py
- **测试**: Pytest、模拟、固定装置

## 关键规则
- 可变默认参数陷阱：`def f(items=[])` 需改为 `items=None` 然后 `items = items or []`
- 身份与相等检查：`is` 检查身份，`==` 检查相等
- 列表修改迭代：使用副本 `for x in list(items):`
- GIL限制：CPU密集任务使用多进程
- 异常处理：避免裸露 `except:`，使用 `except Exception:`
- 编码问题：始终指定 `encoding='utf-8'`

## 安装方式
```bash
# 确保Python环境安装
python3 --version

# 推荐虚拟环境
python3 -m venv myenv
source myenv/bin/activate  # Linux/Mac
myenv\Scripts\activate     # Windows

# 安装常用包
pip install -r requirements.txt
```

## 推荐安装评估

### 本地部署
**适用场景**:
- 个人开发者、小型项目
- 学习Python编程
- 本地脚本开发
- 个人工具开发

**优势**:
- 完全本地化，无网络依赖
- 开发环境配置简单
- 即时反馈，快速迭代
- 学习成本低
- 资源占用少

**ECS部署**
**适用场景**:
- 大型Python项目团队
- 多人协作开发
- 企业级应用开发
- 微服务架构项目

**优势**:
- 统一的开发环境
- 团队协作功能完善
- 代码管理和版本控制
- 自动化测试和CI/CD
- 依赖管理标准化
- 性能监控和优化

**选择建议**: 个人学习和小项目推荐本地部署，专业团队和大型项目选择ECS部署。
