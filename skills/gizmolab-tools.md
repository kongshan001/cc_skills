# GizmoLab Tools

> 区块链开发者工具

## 概述

**Slug:** gizmolab-tools  
**Owner:** gizmo-dev  
**版本:** 1.0.0  
**更新时间:** 2026-02-25

## 功能列表

- 免费区块链开发者工具
- Ethereum 工具：合约 UI 交互、交易解码器、ENS 查找、钱包
- Solana 工具：代币创建、铸造、快照、交换
- Web3 UI 组件：无gas NFT铸造、链选择器、加密产品卡片等

## 使用场景

当构建 dApp、智能合约交互或区块链操作时使用。

## 安装方式

```bash
clawhub install gizmolab-tools
```

| 场景 | 推荐 | 说明 |
