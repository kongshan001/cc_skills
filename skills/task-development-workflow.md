# Task Development Workflow

## 技能描述

TDD优先的开发工作流，包含结构化规划、任务追踪和基于PR的代码审查。适用于需要规范化开发流程的 软件项目。

## 核心功能

- TDD优先的开发流程
- 结构化任务规划
- 任务进度追踪
- PR-based代码审查流程

## 安装方式

```bash
clawhub install task-development-workflow
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐ 较少使用

**理由**: 专注于开发工作流和代码审查，适合本地开发团队使用。需要Git仓库配合使用。
