# OpenClaw Cursor Agent

## 技能描述

通过 OpenClaw 工具管理长时间运行的 Cursor CLI 编码任务，使用 tmux 会话支持。用于用户要求启动持久编码任务、检查任务状态时。

## 功能列表

- Cursor CLI 管理
- tmux 会话控制
- 持久任务运行
- 状态检查

## 安装方式

```bash
clawhub install openclaw-cursor-agent
```

## 推荐安装评估

| 环境 | 推荐度 | 说明 |
|------|--------|------|
| 本地 | ⭐⭐⭐⭐⭐ | Cursor 用户必备 |
| ECS | ⭐⭐⭐⭐ | 远程开发 |

## 适用场景

- 长时间编码任务
- Cursor 集成
- 远程开发管理
