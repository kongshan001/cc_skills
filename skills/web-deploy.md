# Web Deploy

## 技能概述

- **Slug**: web-deploy
- **名称**: Web Deploy
- **作者**: cmanfre7
- **版本**: 1.0.0
- **更新时间**: 2026-02-26

## 背景需求

网站和 Web 应用需要部署到各种平台，需要一个统一的部署工具来管理不同平台的部署。

## 目标

Build, preview, and deploy websites, web apps, and APIs using Vercel, Railway, GitHub Pages, or local Canvas environments.

## 功能列表

- Vercel 部署
- Railway 部署
- GitHub Pages 部署
- 本地 Canvas 环境
- 预览构建
- 多平台支持

## 安装方式

```bash
clawhub install web-deploy
```

## 推荐安装评估

- **本地开发**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS 服务器**: ⭐⭐⭐⭐ 推荐

**理由**: 支持多个主流部署平台，适合有多平台部署需求的开发者。

## 优缺点分析

### 优点

1. 多平台支持
2. 统一部署接口
3. 预览功能
4. 灵活的部署选项

### 缺点

1. 各平台配置不同
2. 学习曲线
