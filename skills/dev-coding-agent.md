# Development Coding Agent

> 增强型编码开发 Agent

## 技能描述

面向开发工作流的增强型编码 Agent，在构建功能、修复 bug、代码重构方面进行了优化。与 OpenCode 集成良好。

## 核心功能

1. **功能构建** - Feature 开发
2. **Bug 修复** - Bug fixing
3. **代码重构** - Code refactoring
4. **OpenCode 集成** - 与 OpenCode 无缝集成

## 安装方式

```bash
clawhub install dev-coding-agent
```

## 推荐安装评估

| 场景 | 推荐度 | 说明 |
|------|--------|------|
| 本地开发 | ⭐⭐⭐⭐⭐ | 开发者核心技能 |
| ECS/服务器 | ⭐⭐⭐⭐⭐ | 团队开发环境 |
| 个人项目 | ⭐⭐⭐⭐⭐ | 提升开发效率 |

### 适用人群

- 所有级别开发者
- 需要 AI 辅助开发的团队
- OpenCode 用户
