# 游戏数值策划 - Game Numeric Design

## 技能描述

游戏数值策划技能 - 基于《游戏数值设定入门》方法论，提供数值设计公式、经济系统、成长曲线等工具

## 功能列表

- 数值设计公式
- 经济系统设计
- 成长曲线规划
- 数值平衡工具

## 安装方式

```bash
clawhub install game-numeric-design
```
