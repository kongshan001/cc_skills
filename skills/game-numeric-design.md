# 游戏数值策划 (game-numeric-design)

## 背景需求

游戏数值策划需要系统的设计公式、经济系统、成长曲线等参考依据，入门者缺乏方法论指导。

## 目标

提供基于《游戏数值设定入门》方法论的工具集，辅助数值设计和平衡调整。

## 设计方案

- **核心内容**:
  - 数值设计公式
  - 经济系统建模
  - 成长曲线设计
  - 数值平衡工具

## 本地部署

```bash
clawhub install game-numeric-design
```

## 效果展示

- 系统化数值设计参考
- 公式与模板工具
- 入门到进阶指南

## 优缺点分析

✅ 优点:
- 方法论系统完整
- 工具化程度高
- 适用多种游戏类型

❌ 缺点:
- 偏向理论，需实践
- 特定类型游戏可能不适用

## 落地过程

1. 安装技能
2. 查询数值设计问题
3. 应用对应公式和工具
