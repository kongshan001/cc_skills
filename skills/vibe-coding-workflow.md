# Vibe Coding Workflow

> Vibe Coding 工作流

## 技能描述

结构化的 5 阶段 AI 辅助开发工作流：需求 → 架构 → 代码生成 → 调试 → 迭代。当用户提到 "vibe coding"、"AI 开发" 时触发。

## 核心功能

1. **需求分析** - Requirements phase
2. **架构设计** - Architecture phase
3. **代码生成** - Code generation
4. **调试** - Debugging
5. **迭代优化** - Iteration

## 安装方式

```bash
clawhub install vibe-coding-workflow
```
