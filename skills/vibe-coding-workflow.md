# Vibe Coding Workflow

## 技能描述

结构化的5阶段AI辅助开发工作流（需求→架构→代码生成→调试→迭代）。当用户提到"vibe coding"、"AI开发"或需要完整开发流程时使用。

## 核心功能

- 5阶段结构化工作流
- 需求分析
- 架构设计
- 代码生成
- 调试和迭代

## 安装方式

```bash
clawhub install vibe-coding-workflow
```

## 推荐安装评估

- **本地开发环境**: ⭐⭐⭐⭐⭐ 强烈推荐
- **ECS/服务器**: ⭐⭐⭐ 适用

**理由**: 通用型开发工作流，本地使用效果最佳。
